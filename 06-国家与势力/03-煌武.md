# 煌武

## 文化基调

- **尚武**：习武是日常美德，但并不意味着武人垄断政权
- **崇直尚言**：敢言、敢谏是社会认可的价值观，不分出身
- **实用主义**：制度和职位不是为了装饰，而是为了解决问题、维持秩序

## 中央权力结构

整体采用**"一相三部六院"**的金字塔形权力架构。皇帝为最高决策与统筹者，集权度远超帝国。

### 一相

**相国**：皇帝首席辅政大臣，统筹协调三部六院事务。对皇帝负责，也接受监察司监督。

### 三部

| 部门 | 职能 |
| ------ | ------ |
| 军部 | 军政事务、军事部署、战时指挥辅助 |
| 政部 | 国家政策制定与施行，收集地方反馈（含部分进谏功能） |
| 吏部 | 人事任免、考核与户籍管理 |

### 六院

| 院 | 职能 |
| ------ | ------ |
| 法院 | 司法审判、律法修订 |
| 工院 | 工匠、军械、交通、水利、工程建设 |
| 礼院 | 礼仪、祭祀、外交典礼 |
| 学院 | 教育、学术研究、人才培养 |
| 廩院 | 粮仓、税收、漕运与储备 |
| 商院 | 商业、关市、贸易与物资流通 |

## 双重监督

### 监察司（自上而下）

直接听命于皇帝，但有权否决政令（象征性制衡）。成员由三部六院各派代表组成，确保多领域专业性。

职责：
- 监督三部六院与地方官员
- 处理重大贪腐与失职案件
- 理论上连皇帝也在其监察范围内（虽无罢免权）
- 监察司本身也接受谏官监督

### 谏官制度（自下而上）

独立于行政与军权，唯一职能是进谏、上达民意。

- **来源**：全民选举——基层谏官由民众选出，层层推选更高一级谏官，直至可面见皇帝
- **任期制+可连任**：只要继续当选即可连任，选举周期短，迫使谏官保持与基层的联系

### 制衡模式

- 皇帝依靠监察司和谏官制约士族
- 士族通过在三部六院中的席位保持影响
- 谏官选举制让底层声音可以直达皇帝，避免民意被士族与官僚阶层阻隔

## 庚辰

煌武境内的[巨龙](../03-龙类设定/01-龙类分类.md)，遭政变者勾结外部势力袭击后陷入沉睡，煌武短期动荡。庚辰是煌武政权的支柱/象征。

# 煌武（Huangwu）

「煌照天下，武定四方」——位于瓦勒瑞姆大陆遥远东方的国家，拥有毫不逊色于帝国的辽阔领土和独特文化。

与西方帝国依靠血脉传承不同，煌武沿袭着类似禅让的制度，统治者通常由民众认可的德才兼备之人出任。统治者名义上受命于天，但仍受制于[道](#信仰)——道被认为是一切事物运行和变化的规律，昏庸无能的统治者会被认为是"失道寡德"之人，继而被推翻。

煌主有多种称谓，如"天命煌主"或"君上"。严格来说煌主只是被民众选出来照顾天下苍生的人，但任何一位合格的煌主，都拥有在煌武的土地上一呼百应的号召力——当之无愧的民众所向、天命所归。

## 文化基调

- **尚武**：习武是日常美德，但并不意味着武人垄断政权
- **崇直尚言**：敢言、敢谏是社会认可的价值观，不分出身
- **实用主义**：制度和职位不是为了装饰，而是为了解决问题、维持秩序

## 中央权力结构

整体采用**"一相三部六院"**的金字塔形权力架构。煌主为最高决策与统筹者，集权度远超帝国。

### 一相

**相国**：煌主首席辅政大臣，统筹协调三部六院事务。对煌主负责，也接受监察司监督。

### 三部

| 部门 | 职能 |
| ------ | ------ |
| 军部 | 军政事务、军事部署、战时指挥辅助 |
| 政部 | 国家政策制定与施行，收集地方反馈（含部分进谏功能） |
| 吏部 | 人事任免、考核与户籍管理 |

### 六院

| 院 | 职能 |
| ------ | ------ |
| 法院 | 司法审判、律法修订 |
| 工院 | 工匠、军械、交通、水利、工程建设 |
| 礼院 | 礼仪、祭祀、外交典礼 |
| 学院 | 教育、学术研究、人才培养 |
| 廩院 | 粮仓、税收、漕运与储备 |
| 商院 | 商业、关市、贸易与物资流通 |

## 双重监督

### 监察司（自上而下）

直接听命于煌主，但有权否决政令（象征性制衡）。成员由三部六院各派代表组成，确保多领域专业性。

职责：
- 监督三部六院与地方官员
- 处理重大贪腐与失职案件
- 理论上连煌主也在其监察范围内（虽无罢免权）
- 监察司本身也接受谏官监督

### 谏官制度（自下而上）

独立于行政与军权，唯一职能是进谏、上达民意。

- **来源**：全民选举——基层谏官由民众选出，层层推选更高一级谏官，直至可面见煌主。出身不限，乞丐、农夫、商人皆可参选
- **任期制+可连任**：只要继续当选即可连任，选举周期短，迫使谏官保持与基层的联系
- **人身保护**：享有最高等级的保护，敢于直言而无性命之忧
- **唯一禁令**：不得上报虚假信息，否则撤职并追责

### 制衡模式

- 煌主依靠监察司和谏官制约士族
- 士族通过在三部六院中的席位保持影响
- 谏官选举制让底层声音可以直达煌主，避免民意被士族与官僚阶层阻隔

## 信仰

煌武不推崇西方式的神明信仰，而是坚信一切自有**「道」**安排。道并非外在的神明，而是人们对世界运行规律的认知——如同自然规律或真理。在这个世界的唯心机制下，「道」也因此真正成为了人们主观认知中存在的实体。

道以两种方式影响煌武之人：

- **因果**：人们对善恶最朴素的认知，也是对自然规律的总结——万事必有其因果，看似偶然背后全是必然。西方向神立誓，东方向天。煌武人极为忌讳以类似预言的方式窥探因果
- **德**：德是道之得，道之用。道是规律和逻辑，德是道的具体表现。人们的言行遵循道，以为有德之人。类似于西方修士遵循戒律才能展现奇迹，煌武顺应道的人也可以展现超凡力量。但与西方戒律不同，德更像是一种普遍价值观——不强制约束，但失德则寡助

详见[煌武的炁与功法](../02-魔法体系/04-煌武的炁与功法.md)。

## 庚辰

煌武的图腾是**应龙**。按帝国学者的定义，应龙属于[巨龙种](../03-龙类设定/01-龙类分类.md)（四足双翼，拥有高度智力和强大力量的个体特化龙类）。

应龙的名字是**庚辰**。在过去数千年的时间里，它一直以圣兽的身份受到煌武人民的崇拜和敬畏，是煌武政权的支柱与象征。后遭政变者勾结外部势力袭击而陷入沉睡，煌武因此陷入短期动荡。

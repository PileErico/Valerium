# 帝国

## 中央权力结构

**议会**：[第一次内乱](../05-历史与时间线/04-征服战争与帝国建立.md#第一次内乱)后，其他龙裔不希望再有人如温德林般完全掌权，从而建立了议会制度。全员[龙裔](../03-龙类设定/02-龙裔起源.md)（统治家族），大事开小会——这种排他性的决策方式让地方领主极度不满，因为他们被帝国中央的政治排斥在外。

已知席位（第二次内战前后）：

| 席位 | 人物 | 备注 |
| ------ | ------ | ------ |
| 第一席 | 皇帝 | 幕后推动血宴 |
| 第二席 | [安德烈·欧恩斯坦](../09-角色设定/03-欧恩斯坦家族.md) | 总帅（军事最高领袖），[血宴](../05-历史与时间线/06-第二次内战（血宴）.md)中被杀 |
| 第三席 | [艾德琳·欧恩斯坦](../09-角色设定/03-欧恩斯坦家族.md) | 外交官，血宴总导演 |
| 第四席 | [洛德·欧恩斯坦](../09-角色设定/03-欧恩斯坦家族.md) | 安德烈提拔的军派龙裔，被派往北境 |
| 第五席 | [维克·欧恩斯坦](../09-角色设定/03-欧恩斯坦家族.md) | 从不露面，魔导装备开发者 |
| 第六席 | [泽娜·欧恩斯坦](../09-角色设定/03-欧恩斯坦家族.md) | 安德烈用龙血制造的生物兵器 |

**内阁**（中央官员）：皇帝 → 首席（辅政）→ 总帅（统军）→ 军务（调度）→ 财政（收支）→ 司法（律法）→ 外交（使节）→ 内务（总管）

## 爵位体系

| 等级 | 名称 | 说明 |
|------|------|------|
| 1 | 皇帝 | 最高统治者 |
| 2 | 公爵/领王/大公 | 行省级，囊括多个郡。大公特指半独立势力 |
| 3 | 边爵（侯爵） | 戍卫边疆的贵族 |
| 4 | 郡爵（伯爵） | 管辖具体郡县 |
| 5 | 堡爵（子爵） | 控制具体据点 |
| 6 | 臣爵/宫爵（男爵） | 无领地宫廷贵族 |
| 7 | 骑士 | 无可继承头衔 |

## 帝国学院

三层结构：
- 下层：通识教育，面向普通人（贵族有私教不上学）
- 中层：优秀者进修魔法等进阶知识
- 上层：最有天赋者前往帝国大图书馆，与资深老魔法师钻研课题

帝国大图书馆：历代领袖喜欢收集各种产物（泽娜、[巫妖湿件](../05-历史与时间线/07-巫妖灾害与北境独立.md)等都是"藏品"）。

## 帝国投资魔法的原因

[龙裔和龙骑士](../03-龙类设定/02-龙裔起源.md)是小规模精锐，大规模战争中作用有限。[艾瑟利亚](../05-历史与时间线/03-七冠时代.md)证明了魔法潜力，帝国以其为戒循序渐进发展。

## 巡回法庭

为推行统一律法建立的流动仲裁团体（教士+学士+宫廷骑士）。效率低导致空窗期，催生了[赏金猎人](../07-组织与职业/02-赏金猎人.md)的定向委托。

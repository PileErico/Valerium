# 九骑士与雷加利亚建国

## 雷加利亚开国九骑士

九位骑士各自代表一种品德，构成了雷加利亚骑士精神的基石。

| 骑士 | 品德 | 品德含义 | 简介 |
| ------ | ------ | ------ | ------ |
| 阿德里克/Adric（无冕者） | 崇高/理想 | 骑士最为纯粹的精神高洁 | 成熟的青年，有着纯粹理想并以此坚定前行。团队核心，雷加利亚最初的领袖和开国者 |
| 兰妮娅·玛格诺利亚/Lannia Magnolia | 执着/坚定 | 骑士应遵循自身主观意愿而前进的必然 | 身着男装的年轻女性，言行大方中性，为达目标不择手段。初心是解放对弱势群体（尤其女性）的压迫 |
| 哈立德·阿尔·扎希尔/Khalid al-Zahir | 不屈/顽强 | 骑士不论面对何种困境都永不放弃的坚韧 | [阿尔·姆哈](01-远古时代.md)异域的酋长，带领子民反抗蛇人但寡不敌众，选择带头撤离故土西迁。为让难民有一席之地而加入，本质是利益交换但初衷是拯救人们 |
| 欧文/Eoghan | 无畏/勇敢 | 骑士必将直面任何危险而不退缩、不犹豫的勇气 | 北地部落勇士，被施加诅咒导致力量强大但生命短暂，为让有限的生命更有价值而投身崇高理想 |
| 维尔南/Vernan | 理性/智慧 | 骑士应谨慎周全地考虑，暴力从来不是唯一的解法 | 魔力的探究者，希望在战斗中实践理论。集体认同感弱，但习惯性地试图理性约束他人的过激举动 |
| 【未命名】 | 责任/担当 | | 年长的战士，沉默寡言但值得信赖，经验最为丰富。因受到所有人尊重而拥有接近领袖的话语权 |

## 天命王权

核心贵族和皇室全是九骑士后裔。民众对统治阶级的信赖通过[唯心机制](../02-魔法体系/03-神术与奇迹体系.md)转化为真实力量（命运加护），但领袖也被束缚——必须成为民众期望的样子，不能拒绝责任。功绩最终归于"天命"叙事。

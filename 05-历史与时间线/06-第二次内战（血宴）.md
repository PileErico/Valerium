# 第二次内战（血宴）

## 背景

皇帝（第一席）不甘被议会架空，渴望中央集权。

## 阴谋三层嵌套

**表层（阳谋）**：[艾德琳](../09-角色设定/03-欧恩斯坦家族.md)联络各地不满的地方领主，许诺外部起兵+内部政变的"宏大叛乱"。目的：动员炮灰。

**核心（阴谋）**：向宫廷内应透露泽娜这张底牌，煽动他们在宴会上刺杀[安德烈](../09-角色设定/03-欧恩斯坦家族.md)。目的：用消耗品引爆事件。

**最终（战略）**：借叛军除安德烈，借安德烈清除激进/投机派贵族，皇帝以"平叛"姿态收场集权。

## 血宴之夜

1. 激进派暴起攻击安德烈 → 安德烈徒手屠灭
2. 投机派见势不妙转攻泽娜 → 泽娜龙人化屠灭
3. 艾德琳在酒/菜中下毒（酒中明毒为诱饵，菜中暗毒为真手）
4. 安德烈中毒后与泽娜死斗
5. 安德烈放弃反杀，抚摸泽娜头顶说"我很抱歉"，靠在她肩上死去

## 安德烈的选择

早已洞悉一切但不阻止。战友亲人百年前全部死去，守护帝国两百年已疲惫。认为年轻人必须脱离庇护才能成长。需要一场盛大的落幕。

## 战后

- 皇帝封锁消息，将事件定性为叛乱
- 向铎利克发诏书邀其出任新总帅
- 洛德被定义为叛党核心
- [北境](../06-国家与势力/05-北境.md)借机独立（奥列克主导）
- 泽娜离开德拉科尼斯，开始流浪

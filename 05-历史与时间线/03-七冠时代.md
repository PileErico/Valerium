# 七冠时代

## 七冠的本质

帝国建立前，人类世界七个最强割据势力形成的政治格局。因安塔利亚被塔克兰殖民的前车之鉴，各势力主动休战，互相承认统治正当性。

## 七冠成员

| 名称 | 核心理念 | 结局 |
|------|---------|------|
| 德拉科尼斯 | 龙之力量、征服、武威 | 统一者，建立帝国 |
| 阿奎莱亚 | 商业、契约、共治（贝洛+莱巴赫双家族，一冠名额） | 纳入帝国，商业传统保留 |
| 奥达尼亚 | 秩序、阶级、责任（议院+骑士团+审判庭） | 衰败，被雷加利亚吞并 |
| 艾瑟利亚 | 求知、探索、魔法研究 | 因魔药滥用覆灭，遗产被帝国学院继承 |
| 维里塔利亚 | 民主、公民责任（外号：杰米尼亚/双生之国） | 分裂为科鲁维亚（南，集权）和科莱帕齐亚（北，民主），后被征服 |
| 雷加利亚 | 天命王权、英雄血脉 | 吞并奥达尼亚，最终被帝国征服 |
| 格莱德尼亚 | 与自然共生、德鲁伊结社、部落联盟 | 迅速衰落，部分传统延续 |

## 七冠之外的势力

- **阿戈隆德**：七冠时期存在但非冠级，被帝国征服
- **安塔利亚/埃特尼亚**：西南半岛，太边缘孤僻不在七冠体系内
- **塔克兰**：海外殖民者，来自西南群岛
- **厄里恩特**：弱小缓冲地带，名义上属西部联合

## 命名规则

与德拉科尼斯同级的领地（日后降为公爵领）用拉丁语命名。德拉科尼斯以日耳曼语为主。

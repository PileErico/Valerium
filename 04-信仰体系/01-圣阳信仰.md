# 圣阳信仰

## 概述

[帝国](../06-国家与势力/01-帝国.md)主流信仰。核心是人们对光明、正义、力量的追求。萌芽于战争，源自对强大力量的渴求和对不公世道的愤怒，因此武力与虔诚同等重要。

神明：索利斯（Solis）——至上，无性别，绝对公正且怜悯众生。实际上是无自我意识的概念黑箱。

权能：力量、公正、疗愈。可以治疗伤口、辨识谎言，但不能降雨肥田。（参见[神术与奇迹体系](../02-魔法体系/03-神术与奇迹体系.md)）

其中公正是圣阳独有的权能——太阳普照万物，绝对无私。信徒对抗的并非"强者"，而是"不公"这一事实本身。即便强者受到不公待遇，教士同样会干涉。公正的定义简而言之：**得其所应得**。

**赐福与诅咒**：修道者立誓时获得圣阳祝福——持有者不易心生恐惧，因光明必将驱逐黑暗。背弃信仰的修道者则受到诅咒——不再能被圣阳奇迹疗愈。这看似轻微，实则严重：因为奇迹的存在使得帝国的医学研究长期停滞不前，失去奇迹疗愈几乎等同于放弃治疗。

## 教派分裂

- **新派**：追求公正与平等（后天概念，主观定义正义）
- **旧派/保守派**：崇尚光明与纯粹（本能趋向，绝对纯粹和客观）

## 教会组织结构

### 信众与神职

- **平信徒（Laity）**：单纯信仰神明的普通人，无法使用奇迹。基层教派活动离不开平信徒的援助，不少普通教士也是平信徒出身，单纯出于好心参与教会事务
- **教士（Priest）**：信仰强烈到足以引发奇迹者。通常不参与争斗，是教会基层且占比最大的群体。主要从事社区服务等基层工作
- **修士**：与普通教士有明确区分——需要进入修院专业学习、修行，身体力行践行教义。毕业后一般去圣阳的医院治疗伤员、驱逐病痛，也处理民间纠纷——很多时候人们并不完全信任贵族和官吏，即便是判刑也要求有公正的圣阳修士在场
- **武装修士**：真正意义上的神职战士（如[艾尔里奇](../09-角色设定/01-主角团（六人）.md)）。一般只与袍泽接触，任何时候都应遵循戒律

### 层级

| 层级 | 对应范围 | 说明 |
| ------ | ------ | ------ |
| 教士/修士 | 基层 | 社区服务、医疗、纠纷调解 |
| 教长 | 小聚落 | 资历深的教士或退休修士/武装修士 |
| 教区主教 | 城市/定居点 | |
| 大主教 | 地区级（如科鲁维亚主教） | 教派内部事务由各领大主教开会决定 |
| 枢机主教 | 无辖区 | 教会最高级神职之一 |

武装修士系统的上级是团长和修院院长（有时为同一人），最高领袖是**大团长（Grand Master）**，由内部选举产生。

### 教会决策层

枢机主教 + 大主教 + 大团长 + 部分执行官共同决断教会内部主要事务。

### 特殊职位

- **执行官（Executor）**：独立于教会常规体系，判断不受教义束缚。不从属于裁判所
- **裁判所（Inquisition）**：针对异端审判，遵循明确的戒律和教义

### 武装团体

圣阳信仰萌芽于战争，武力与虔诚同等重要。擅长战斗的教士自发聚集组成军事修会或宗教骑士团，接受地方主教指挥但无从属关系。

特殊武装团体：
- **日冕骑士团**：与裁判所关系密切，不必听命于地方主教。并非常见的教士——不具备占卜和赐福能力，但可以直接借用索利斯的**权能**进行战斗。能令武器和甲胄被炽热的阳炎包裹而不伤及自身，更进一步可将热量内敛以增幅肉体能力，但负荷状态下身体会不断升温而无法持久
- **宗座近侍**：驻守核心教区、保护大主教的近卫精锐，由枢机主教或支团长级别的神职人员带领

## 圣城奥利图尔与祝圣武器

**圣城奥利图尔**：位于遥远东方的圣阳教会圣地。

**淬阳仪式**：将武器置于奥利图尔的祭坛顶端进行祝圣。经过仪式的武器从此在阳光下无法被摧毁，且具备对邪崇之物的驱散效果。

城市沦陷后再无祝圣武器诞生——早在塔克兰战争结束以前教会就失去了对圣城的掌控，战后远征军光复时只剩废墟。即便帝国重建了同等乃至更大规模的祭坛，仍然无法进行仪式。大概只有那个地方才是被神所应许的。

**祝圣武器的管理**：已存在的祝圣武器因其特性得以长期保存，被升格为**圣器**。由地位较高的骑士团成员持有，但成员并不拥有武器本身，只是被授予使用权。非外勤时期武器统一存放在圣殿中。只有在极严峻的事态下，才会临时授权教会执行官使用。

## 帝国学院与教会的张力

帝国魔法师是唯物主义科学家，试图研究和解构神明本质。引发与神职者/信徒的冲突。中央和稀泥，压制学院不要明目张胆研究。

学院高层经过几代人暗中研究，已对信仰本质猜得八九不离十，但结论太颠覆被封存为"仅作为一种可能性"。

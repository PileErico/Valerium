# 以太与魔力本质

## 以太（Aether）

以太是承载魔力的基本粒子。

**来源**：以太粒子来自该星球的一颗卫星。如同太阳输出光热，该卫星输出高活性的以太粒子。

> 注：旧设定中以太来自"以太位面"（另一个能量位面），已废弃。现设定以太为本星系内的物理现象。

## 以太能级体系

以太拥有从0到5的能级：

| 能级 | 状态 | 说明 |
|------|------|------|
| 0 | 基态 | 施法者利用完能量后的状态，魔力为零 |
| 1 | 接近基态 | 极稀薄魔力 |
| 2 | 亚稳态 | 普遍情况之一 |
| 3 | 弱激发态 | 普遍情况之一 |
| 4 | 强激发态 | 普遍情况之一 |
| 5 | 全激发态 | 最高能级 |

**关键规则**：
- 空间中的以太并非默认基态，而是根据环境和空间不同拥有不同能级（通常2-4）。
- 基态（0）是施法者利用完以太中的能量后的状态。
- 连续施法后，区域内以太被消耗降至低能级，导致后续施法困难（魔力浓度下滑）。

## 魔力与灵魂

- 人类的灵魂是魔力的载体。自然界的以太粒子会被灵魂吸收，当灵魂被魔力填满后，多余的魔力会溢出并消散。
- 施法者通过冥想等方式恢复魔力。

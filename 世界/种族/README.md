# 种族

这个世界的智慧生物并不全是自然演化的产物。有些来自漫长的进化，有些则诞生于远古文明的实验与意外。它们之间的差异不仅体现在外貌和能力上，更深层的区别在于构成身体的基础物质——碳基的血肉，或是以太粒子的凝聚。

---

### [人类](人类.md)

足迹遍布世界各个角落，适应力极强，善于学习与传承。虽生理强度不占优势，却凭不断进取的特性成为世界上最具影响力的存在。大陆东西两端分别由[帝国](../文明/帝国/政治与制度.md)与[煌武](../文明/煌武/煌武.md)主导。

### [精灵](精灵.md)

世俗所知的"精灵"实际涵盖两种存在：来自上个世代、真正永生不老的**真·精灵**，以及真·精灵与人类混血后逐代稀释的**半精灵**。半精灵因寿命长、耐用而在某些地区沦为奴隶市场的商品。

### [鬣狗人](鬣狗人.md)

与鬣狗极为相似的异形种族，母系社会，被饥饿感驱使。曾被兽人奴役为下位种族，在此过程中驱动力从"饥饿"转变为"憎恶被奴役"。供奉名为"[The Hunger](../信仰/其他信仰.md#鬣狗人---the-hunger)"的神明。

### [鱼人](鱼人.md)

诞生于[灵渊](../信仰/灵渊信仰.md)的种族，孵化时会经历各种突变。近岸的两栖鱼人已与深海同族踏上完全不同的进化道路，智力稳步提高，甚至有个体在费什海姆成为正式冒险者。

### [爬行种](爬行种.md)

阿尔·姆哈的远古文明种族，分为龙人（领袖）、蛇人（神职）、蜥蜴人（底层）三个种姓。现存的蜥蜴人大多源自被蛇人献祭仪式转化的人类，以商人身份出没于大陆各地。

### [兽人](兽人.md)

[卡·文图斯](../文明/其他/卡·文图斯.md)的主体种族，比鬣狗人更先进。【设定待推进】

### [矮人](矮人.md) 【待定】

存在于历史记录中，狩猎地龙、与帝国交涉，使用文字形式的符文。最终形态未确定。

### [龙裔](龙裔起源.md)

并非独立种族，而是通过汲取古龙之血蜕变的人类。德拉科尼斯的猎龙传统、贝希摩斯之血、温德林的重生仪式——龙裔的诞生改变了整个大陆的力量格局。

# 精灵

世俗所知的"精灵"实际上涵盖了两种截然不同的存在。

对一般人而言，精灵是生活在林地中、有着优秀魔力适性的长生种，性情冷漠，不愿参与世俗纠纷。可能受[德鲁伊文化](../信仰/自然信仰与德鲁伊.md)影响，精灵同样没有记录历史的习惯，无从追溯其起源。精灵似乎只在瓦勒瑞姆大陆西部的大森林及附近出没，对遥远的东方人类而言是完全陌生的存在。

精灵持有特殊的货币——**金鹰**和**银狼**（比值约60:1），因工艺和材质特殊而被帝国商人认可，价值略高于帝国币。但这些货币并非出自林地精灵之手，精灵也不将其视为一般等价物。

## 真·精灵（True Elves）

来自"上个世代"的存在，真正的永生不老。从上个世代至今已生活了无法计算的岁月。正常生物早就精神崩溃，实际上大部分真·精灵都选择了自杀。仍活着的都是靠某种执念或追求支撑。

四处流浪，在各个文明中零星分布。数量极少。

## 半精灵（世俗所称的"精灵"）

真·精灵与人类的后代。相比人类寿命极长，但相比真·精灵是劣化的。

半精灵族群的形成：某位真·精灵突发奇想四处交配，试图用这种方式创造全新族群。

精灵血统逐代稀释：真·精灵 → 与人类混血生出半精灵 → 半精灵继续与人类混血 → 后代仍被通称"半精灵"但血统越来越稀薄。血统稀薄到常见不珍贵的程度，社会上不会当珍稀种族保护，反而因寿命长、耐用而在某些地区成为奴隶市场的商品（如索伦特）。

# 编年史

> 瓦勒瑞姆大陆西部已知历史事件的时间排列。短篇链接嵌于对应事件旁，可直接跳转阅读。
>
> **纪年**：帝国历（IE），以征服战争结束翌年帝国建立为元年。征服战争前以 BIE 表示。年份为大致推算，日后细究。

---

## 远古时代

| 事件 | 详情 |
| ------ | ------ |
| 上个世代 | 真·精灵来自此时代。曾经的人类比现在更加发达 |
| 贝希摩斯倒下 | 始源兽龙因重力无法支撑而倒下，选择化整为零。欧恩斯坦先祖窃取贝希摩斯之血并封存 |
| 爬行种文明 | 阿尔·姆哈的种姓制度（龙人/蛇人/蜥蜴人）。蛇人预言冰期，修建地下建筑长眠，实则偷天换日 |
| 冰期结束 | 蛇人醒来，蛊惑苏丹转信蛇神，将阿尔·姆哈人转化为蜥蜴人奴隶。哈立德·阿尔·扎希尔反抗，带领子民西迁 |
| 构装之民 | 以火山为核心的工业文明（Machinatus）。因矮人狩猎地龙引发地震，被岩浆掩埋 |

## 七冠时代（~BIE 200 — BIE 16）

| 事件 | 详情 |
| ------ | ------ |
| 九骑士建国 | 阿德里克率领九骑士建立雷加利亚，天命王权体系确立 |
| 七冠并立 | 德拉科尼斯、阿奎莱亚、奥达尼亚、艾瑟利亚、维里塔利亚、雷加利亚、格莱德尼亚相互承认 |
| 塔克兰入侵 | 来自西南群岛的殖民者入侵安塔利亚 |
| 艾瑟利亚覆灭 | 魔药滥用损害灵魂，整个文明覆灭。遗产被帝国学院继承 |
| 维里塔利亚分裂 | 同母异父的姐弟分治——北方科莱帕齐亚（民主）、南方科鲁维亚（集权） |
| [厄尔诺屠夫](../../故事/短篇与片段/厄里恩特线03-厄尔诺屠夫（泽兰卫队）.md) | 泽兰卫队镇压厄尔诺叛乱，屠杀后被解散 |
| 德拉科尼斯猎龙传统 | 各猎龙家族以狩猎兽龙、沐浴龙血为传统。兽龙越来越狡诈，迫使德拉科尼斯另谋出路 |
| 温德林发现龙血 | 找到先祖封存的贝希摩斯之血，开始数十年的化龙研究 |
| 重生仪式 | 温德林召集各家族精英参加仪式，暗中先行适应龙血。率先苏醒后杀死所有非欧恩斯坦幸存者。龙裔诞生 |
| [莱瑟的抉择](../../故事/短篇与片段/厄里恩特线01-莱瑟的抉择.md) | 莱瑟15岁被带回厄里恩特。面对父亲的提问做出抉择，罗伊斩杀恩佐 |

## 征服战争（BIE 16 — BIE 1）

| 事件 | 详情 |
| ------ | ------ |
| 温德林发起征服 | 为调集化龙所需资源而统一西方。龙裔的力量打破了七冠的均势 |
| 维里塔利亚被征服 | 科鲁维亚被莱瑟（此时臣服于德拉科尼斯）击垮。科莱帕齐亚战败后北逃，被北地人收留 |
| 莱瑟获封 | 战功显赫，封为科鲁维亚公爵。封米勒为科克兰郡爵 |
| [科克兰的冬日](../../故事/短篇与片段/厄里恩特线04-科克兰的冬日（米勒）.md) | 米勒的内阁议会，埃文·坎德拉来访，圣战号召 |
| 帝国建立 | 七冠降级为公爵领，各地领王名义臣服。征服匆忙，无力直接管制地方 |

## 第一次帝国内战·公爵叛乱（~IE 10-15）

| 事件 | 详情 |
| ------ | ------ |
| 温德林无心统治 | 将一切资源投入化龙研究，对政务不闻不问 |
| 莱瑟发起叛乱 | 暗中拉拢反对派，目标解体帝国、夺回厄里恩特 |
| 各势力介入 | 科莱帕齐亚遗民寻求复辟、塔克兰巨人参战、煌武观察团来看戏、德鲁伊一如既往反帝国、兽人被许以德拉科尼斯土地、安塔利亚支持帝国以求承认 |
| 温德林化龙 | 决战中温德林以战场上所有人的血肉和灵魂为养料完成化龙。龙裔和人类联手对抗巨龙 |
| 巨龙陨落 | 各派系精锐联手杀死巨龙。温德林得偿所愿 |
| 科莱帕齐亚复辟 | 帝国许诺其以自治领形式延续民主政治 |
| 莱瑟的结局 | 没赢没输，保住科鲁维亚、拿回厄里恩特，修建科布伦茨堡垒 |
| 战后休养 | 各方两败俱伤，帝国反而重新稳定。十余年无战事 |

## 帝国时代（IE 15 — IE 206）

| 事件 | 详情 |
| ------ | ------ |
| 议会建立 | 龙裔不愿再有温德林式独裁，建立六席议会制度 |
| 安德烈守护帝国 | 以总帅身份守护帝国两百余年。培养洛德（第四席）、制造泽娜（第六席）、训练铎利克 |
| 巡回法庭 | 为推行统一律法而建立流动仲裁团体。催生赏金猎人定向委托 |
| 开拓热潮 | 魔法深入→生产力提高→人口上升→对土地需求增加。开拓者逐渐转型为冒险者 |
| 德鲁伊衰亡 | 帝国开拓侵入森林→德鲁伊诱导动物袭击→交涉无果→帝国捕杀 |
| 银鹫会成立 | 瓦莱里安整合军事贵族对抗商业联合会 |
| [被知识诅咒的人](../../故事/短篇与片段/帝国01-被知识诅咒的人（赫芬）.md) | 赫芬的学院入学演讲——"魔法师不过是被知识诅咒的人" |
| [沃尔夫冈与贝拉](../../故事/短篇与片段/厄里恩特线02-沃尔夫冈与贝拉.md) | 庄园大火，贝拉之死，沃尔夫冈成为赏金猎人"七指" |
| [积雪与归来](../../故事/短篇与片段/艾登与瑞秋线01-积雪与归来.md)、[冬日练剑](../../故事/短篇与片段/艾登与瑞秋线02-冬日练剑.md) | ~IE 200：瑞秋年幼时期的北境日常 |
| [离别之战](../../故事/短篇与片段/帝国02-离别之战（安德烈与铎利克）.md) | ~IE 206：铎利克拒绝继任第二席，与安德烈对决后离开帝都赴任莫洛恩 |
| 安德里亚斯地下实验 | ~IE 208起：巫妖在北境地下进行数年实验，积累控制和改造尸体技术。瓦诺（无头骑士）驻守地下城 |

### IE ~214-217：发展

> 修出门两年多才遇到亚伦。安德里亚斯完成研究后密集行动。

| 帝国中央 | 北境/巫妖暗线 | 角色们 |
| ------ | ------ | ------ |
| 艾德琳暗中联络地方领主 | 安德里亚斯完成研究，离开地下城 | 修与亚伦相遇结伴（Nasty Duo） |
| | 上山杀死海姆斯（极境山脉雪峰） | [初遇](../../故事/短篇与片段/伊修娜与艾尔里奇01-初遇.md)——伊修娜被教会收留，策反艾尔里奇出走 |

| | 奥列克掌控北境，洛德回到管理层 | [绿骑士](../../故事/短篇与片段/独立线01-绿骑士（莱恩与博鲁特）.md)——莱恩与博鲁特被殴打 |

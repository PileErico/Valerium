# 命名规则

## 派系命名语系

| 语系 | 适用范围 | 说明 |
| ------ | ------ | ------ |
| 日耳曼语 | 德拉科尼斯及帝国核心 | 帝国发源地的语言传统 |
| 拉丁语 | 与德拉科尼斯同级的领地（日后降为公爵领） | 阿奎莱亚、维里塔利亚、雷加利亚等 |
| 英语 | 阿戈隆德等被征服地 | |

## 人名/姓氏含义

### 德拉科尼斯（日耳曼语系）

| 名字 | 含义 |
| ------ | ------ |
| Wendelin（温德林） | 漂泊流浪者 |
| Andre（安德烈） | 战士 |
| Ilse（伊尔莎） | 神（即龙）的誓言——非俯首于神明，而是顺从于血脉中流淌的本能 |

### 阿戈隆德（英语系）

| 名字 | 含义 |
| ------ | ------ |

### 无姓氏角色

| 名字 | 原因 |
| ------ | ------ |
| 亚伦（Aaron） | 平民出身，亚伦甚至不是真名，自己随便起的 |
| 伊修娜（Ishona） | 父亲为保护她没有给她姓氏 |
| 瑞秋（Rachel） | 奴隶出身无姓氏，想跟艾登姓被拒（蛮族姓氏不适合她） |
| 阿尔弗雷德 | 孤儿出身 |
| 塞塔 | 平民出身的游侠骑士，更可能给自己取外号而非姓氏 |

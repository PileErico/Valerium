# 生物

## 以太与生物演化

以太存在于宇宙、星球的时间甚至比生命还早。如同太阳向地表输出光和热，这个世界的[卫星](../施法体系/以太与魔力本质.md)持续向星球输出以太粒子——一种更特殊的能量。所有生物都在以太的照拂下演化，就像现实中所有生物都在阳光下演化一样。

这个世界不存在现实意义上的"普通动物"。正如现实中有的生物高度依赖阳光（变温动物离了太阳就无法维持体温），有的生物对阳光几乎无所谓（深海生物），这个世界的生物对以太的依赖也是如此——程度各异，没有明确界限。

- 有些生物几乎不依赖以太，有没有都一样活，所以看起来和现实中的动物差不多
- 有些稍微利用了一点，但外在表现不明显
- 有些深度依赖以太，进化出了以太驱动的特殊器官和能力，一旦离开高能以太环境就会快速死亡——正如变温动物离开阳光就无法存活，依赖并不意味着强大

以太的高能性使这个世界的活跃度远超现实——文明更替更频繁，进化更快，但能走到最后的很少。因为基于以太这种特殊变数而成立的生物和文明都是脆弱的：没能拥有适配其能力的认知而导致自我毁灭并不稀罕。

对以太的利用是进化的一部分——是本能，是天赋能力，不是魔法。北地的冰巨魔体格强大，但它拥有的魔力大部分被作用于强大的肉体再生能力和保护自身的特殊器官——体内储存超低温液体的小囊可在必要时喷涂低温液体，液体瞬间闪蒸凝结为白雾，正常生物触碰即冻伤，吸入则因呼吸道坏死而直接死亡。这不是魔法，这是它的生态，是进化出来的能力。

人类恰好是演化过程中无数次分叉都避开了以太利用为主的变异，最终保持了不被以太深度塑造的纯粹。但人类不是唯一的这类情况，也有别的生物一路演变下来没有被以太显著改变。人类的独特之处在于知性足够高，由此发展出了主动利用以太的技术——即[魔法](../施法体系/体系总览.md)。

## 怪物

**怪物**不是学术分类，是实用分类——拥有以太驱动的特殊能力、需要专业手段对付的生物。它们的生态极为鲜明突出，数量更少，因为其生理特性同时受到正常生态条件和以太条件的双重制约。

一般来说，会被[冒险者](../文明/帝国/职业/冒险者.md)选为讨伐目标的，都是因以太而进化出特殊能力的生物。

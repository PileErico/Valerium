# 信仰

信仰起源于对自然与未知的敬畏和恐惧。太阳、海洋、大地，都曾产生过对应的信仰。随着魔法与科技水平的提高，大地信仰逐渐被摈弃——农业问题都能被魔法克服。而留存至今的信仰，则在漫长的岁月中不断演变，发展出了各自的体系。

## 泛灵信仰

对自然本身的崇敬，不指向特定的神明，而是对世界万物的敬畏。

- [自然信仰与德鲁伊](自然信仰与德鲁伊.md) — 塞纳诺斯（森林意志的人格化）、德鲁伊传承、森林精魂与绿骑士
- [灵渊信仰](灵渊信仰.md) — 沿海地区的二元信仰，生命温床与海洋的双重面向，堕渊者

## 神明信仰

对特定神明的崇拜。神明并非先于信仰存在的实体，而是信仰的产物——足够强烈且一致的群体认知，能在物质层坍缩为真实存在。

- [圣阳信仰](圣阳信仰.md) — 太阳神索利斯，帝国主流信仰。核心为力量、公正与疗愈。教会组织庞大，分新派（公正）与旧派（纯粹）
- [其他信仰](其他信仰.md) — The Hunger（鬣狗人）、爬行种的蛇神信仰、构装之民的锻造神等

## 信仰的物质化——神格

当信仰附着在真实存在的实体上时，可催生神格，产生有自我意识的半神乃至神。

- **库库尔坎**：羽龙始源龙，数万年供奉后催生神格 → 详见[龙类分类](../生物/龙类分类.md)
- **菲尼克斯/维罗妮卡**：安塔利亚的半神领袖 → 详见[安塔利亚](../文明/安塔利亚/安塔利亚.md)

## 底层规则

信仰之所以能产生真实的力量，根源在于这个世界的底层规则——认知层对物质层的扭曲。

→ [共识（Doxa）](../底层规则/共识（Doxa）.md) — 认知坍缩与奇迹原理

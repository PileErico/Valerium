# 施法体系

魔法分类并非宇宙通用的客观分类，而是各文明基于自身世界观和哲学建立的认知框架。帝国基于元素理论发展出奥术/神术体系，煌武则以"炁"的认知发展出功法体系，但底层本质都是以太能级变化。

# 世界观设定项目

## 项目概述

这是一个原创奇幻世界观的设定整理项目，企划名为**命运长歌（Valerium）**。所有设定文档以中文 Markdown 编写，按语义主题分类存放于仓库根目录下的中文命名子文件夹中。文档之间通过 Markdown 相对链接互相跳转，在 GitHub 上可以像百科一样浏览。

## 当前状态

项目正处于**渐进式重构**阶段。旧的设定文档保留在 `01-世界（旧）/` 和 `04-附录（旧）/` 中作为内容源，新结构在 `01-世界/` 和 `04-附录/` 中以带序号的方式建立，内容一点点搬运和重新整理，而不是原地大改。

## 核心设定要点

### 底层规则

- **真知（Episteme）** — 物质层客观规律
  - **推理（Dianoia）** — 假设推演，构建术式模型（奥术进步的方法论）
  - **直悟（Noesis）** — 对事物本质的直接理解
- **共识（Doxa）** — 认知对物质层的覆写
  - **信念（Pistis）** — 具体认知（如对神明的信仰）
  - **映象（Eikasia）** — 认知导致的物质变化（如奇迹）
- **知性**：跨过某个阈值的生物才能参与共识机制（鬣狗人的浅层认知就足够）

### 生物与以太

- 以太源自本星系一颗轨迹不定的卫星（并非上个世代设定中的"以太位面"）
- 所有生物都在以太环境中演化，依赖程度是连续光谱
- 人类与"正常动物"恰好没走上以太依赖的演化分支
- 怪物是深度依赖以太的进化产物，能力是生理本能而非魔法
- 知性生物中，人类主动利用以太发展出魔法

### 上个世代与星民

- **上个世代**：原子论起步→接近真知→质能转换→以人类志愿者为蓝本用以太粒子复刻造出不朽者→实验失控→覆灭
  - **不朽者（The Eikon）**：世俗通称"真·精灵"，数十个被造出，帝国历仅存不足十个
  - **精灵（Elf）**：某位不朽者一时兴起与人类交配的产物，血统稀释后即为"半精灵"
- **星民（Astralis）**：覆灭于"长周期以太衰减→恐慌→内耗"的古文明，信仰蔚蓝星体（启明 Lucifer / 归终 Noctifer）。其语言被各地学者从遗迹铭文中破译，成为七冠会议上定下的通用外交语言——即**星语（Sermo Astralis）**

### 派系进度

**已有完整设定**：德拉科尼斯/帝国、雷加利亚、维里塔利亚（含科鲁维亚、科莱帕齐亚）、格莱德尼亚、卡·文图斯

**待补充**：阿奎莱亚、艾瑟利亚、奥达尼亚、阿戈隆德、厄里恩特（占位）、煌武、安塔利亚、塔克兰、阿尔·姆哈

### 命名语系

各派系日常使用自己的方言（日耳曼/诺斯/凯尔特/古英语/阿拉伯/汉语等），正式场合用星语。七冠地名全部是星语命名（-ia 后缀等），在七冠会议上定下。

## 工作方式

- 用户口述或讨论设定 → 我整理写入对应文件
- **大块内容/重组由我执行，小改由用户手动进行**
- 新增内容尽量先写入 `01-世界（旧）/` 对应位置，等统一搬运时一起整理
- 标注【待定】的内容表示未最终确认
- 讨论时优先提问和反馈，不直接接受内容——用户希望先碰撞再落笔
- 推送由用户明确要求后执行，不自动推送
- 本文件（CLAUDE.md）随进度变化即时更新

## 互链规范

- 文档顶部统一添加 `[← 返回目录](../README.md)` 导航（层级根据文件深度调整）
- 正文中首次提及其他文档中有详细说明的概念/角色/地点时，使用相对链接
- 文档底部添加 `相关条目` 区域，列出最相关的 3-6 个文档链接
- 新增或修改内容时，同步维护相关文档中的链接
- **GitHub 的 tree 链接（文件夹页面）被 robots.txt 屏蔽，AI 工具无法访问；blob 链接（具体文件）可以访问。所以所有文件夹链接应指向该文件夹下的 README.md 而非裸文件夹路径**

## 对话风格要点（用户偏好）

- 用户希望 AI 在接受内容前先提出问题和反馈，不要直接整理写文档
- 不要过度剪裁用户提供的文本内容
- 不要动不动就推送——推送由用户明确发令
- 单次讨论深入而非宽泛——聊透一个设定再进下一个
- 文档排版需要有节奏，避免扁平的信息堆叠

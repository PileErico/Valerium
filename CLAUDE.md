# 世界观设定项目

## 项目概述

这是一个原创奇幻世界观的设定整理项目。所有设定文档以中文 Markdown 编写，按主题分类存放于 `世界观/` 根目录下的子文件夹中。

## 当前进度

- [x] 从总文档 `worldbuilding.md` 拆分为结构化文件夹和文件（2026-04-03 完成）
- [ ] 待定设定的补充（见 `11-待定与备忘/待定设定.md`）
- [ ] 新增设定的持续整理

## 工作方式

- 用户口述或讨论设定 → 我整理写入对应文件
- 新增内容归入现有结构，必要时新建文件或文件夹
- 标注【待定】的内容表示未最终确认，【旧设定/参考】表示可能被覆盖，【联动】表示非核心设定
- 修改设定时同步更新所有关联文件中的相关内容
- 本文件（CLAUDE.md）随进度变化即时更新

## 设定标注约定

| 标注 | 含义 |
|------|------|
| 【待定】 | 作者尚未最终确认 |
| 【旧设定/参考】 | 可能被后续更新覆盖 |
| 【联动】 | 来自与朋友世界观的联动，非核心设定 |
| 【待补充】 | 框架已定但细节未写 |

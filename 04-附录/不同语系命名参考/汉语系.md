# 汉语系

适用地区：煌武

## 构词规则

### 人名构成

煌武人名遵循汉语传统，姓在前、名在后。姓氏通常为单姓，名一到二字。

**姓**：多取自百家姓体系，反映家族传承。修炼世家常以单姓流传，门派弟子有时以师门之姓为姓。

**名**：含义多与品德、自然、志向相关。修炼者成年后可取**字**（表字），由师长或本人择定，与名相呼应。

**道号/法号**：丹修（道士）入道后由师父赐予道号，取代俗名使用。气修和体修通常不取道号。

### 地名构成

- **-州**：行政区域。如临沧州
- **-城**：城郭
- **-山**：山岳
- **-峰**：山峰（常用于门派所在地）
- **-谷**：山谷
- **-川**：平原/河流
- **-海**：海域/大湖
- **-河/-水**：河流。如沧河（沧水）
- **-阁/-楼/-殿/-宫**：建筑群落，常用于门派驻地命名
- **-门**：门派/山门。如天剑门、青云门
- **-派**：流派
- **-宗**：宗门

**方位命名**：遵循"山南水北为阳，山北水南为阴"的古制。如沧阳——位于沧河北岸。临（濒临）、浦（水滨）、津（渡口）、源（源头）等为常见滨水地名用字。

### 功法/门派命名

- 常用前修饰词：天、地、玄、黄、太、无、真、灵、紫、金、玉
- 常见后缀：诀、法、功、经、术、道、式
- 例：太玄经、紫霞功、天罡诀、无极道

### 招式命名

通常为四字或三字，结构多为主谓或偏正：

- 天外飞仙、万剑归宗、一气化三清
- 排云掌、风神腿、天霜拳
- 破剑式、破刀式

## 已使用的名字

### 河流与湖泊

| 中文 | 类别 | 含义 |
| ------ | ------ | ------ |
| 沧河 | 河流 | 煌武南部主要水系。性质近长江，集中主干。沧，水之青苍深远 |
| 泱水 | 河流 | 煌武中部主要水系。性质近淮河，散开成网、多汊连湖。泱，水面广阔 |
| 天镜 | 内陆湖 | 泱州偏西，永安西渠引水蓄成的人工大湖。湖水如镜 |
| 永安渠 | 人工运河 | 煌武南北大动脉，分永安西渠（入天镜）与永安东渠（抵沧阳） |

### 九州

单字+州，取禹贡式朴素质感。

| 中文 | 位置 | 含义 |
| ------ | ------ | ------ |
| 朔州 | 北境 | 朔，北方。横贯北境，驻军边防 |
| 垠州 | 西北 | 垠，边际。半干旱草原，接卡·文图斯 |
| 岚州 | 西 | 岚，山林雾气。山前丘陵+沼泽，湿润多雾 |
| 嵚州 | 西南 | 嵚，山势高峻。天山南段，沧河源头，圜丘所在 |
| 泱州 | 中央偏南 | 泱，水面广阔。京畿，天枢所在，辖天镜湖 |
| 衍州 | 偏东北 | 衍，水朝宗于海+漫衍平野。泱水诸汊汇集 |
| 浦州 | 东北 | 浦，水滨入海处。泱水入海口 |
| 陉州 | 中东部 | 陉，两山间隘口。南北咽喉，分隔泱沧水系 |
| 沧州 | 东南 | 沧，水之青苍。沧河出海口。全名/雅称"临沧" |

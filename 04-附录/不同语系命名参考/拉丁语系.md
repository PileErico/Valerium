# 拉丁语系

适用场景：星语（Sermo Astralis）命名体系、七冠官方地名、帝国学术/宗教术语

星语是大陆西部的通用外交语言，源自星民（Astralis）遗迹铭文。七冠会议将其采用为正式名称和盟约文书的书写语言。其命名规则在构词法上参考拉丁语。

---

## 构词规则

### 地名构成

七冠地名全部以星语命名。最常见的地名后缀：

| 后缀 | 含义 | 举例 |
| ------ | ------ | ------ |
| **-ia** | 国土/领地/地区 | Draconis → Draconia（德拉科尼斯，龙之领地）、Regalis → Regalia（雷加利亚，王权之地）、Aquileia（阿奎莱亚）、Odania（奥达尼亚）、Antalia（安塔利亚） |
| **-ium** | 地方/场所（中性） | Elysium（艾瑟利亚） |
| **-is** | 属性/归属（名词化后缀） | Draconis（德拉科尼斯，龙的/属龙之地） |
| **-um** | 地点/物（中性） | Metallum（梅塔卢姆） |

地名通常由一个实义词根加后缀构成：

- **Draconis**：draco（龙）+ -is（属…之地）→ 龙之地
- **Regalia**：regalis（王权的）+ -ia → 王权之地
- **Aquileia**：aquila（鹰）+ -eia → 鹰旗之地

### 人名构成

星语人名通常由**本名（Praenomen）+ 族名（Nomen）+ 附加名（Cognomen）**构成。正式场合使用全名，日常可用本名或附加名。

**常见词根**：

| 词根 | 含义 | 举例 |
| ------ | ------ | ------ |
| Luci- | 光 | Lucius、Lucia |
| Val- | 强壮 | Valerius、Valeria |
| Marc- | 战神 | Marcus、Marcellus |
| Aurel- | 金色的 | Aurelius、Aurelia |
| Flav- | 金发/黄色 | Flavius |
| Sept- | 第七 | Septimus |
| Oct- | 第八 | Octavius |
| Quint- | 第五 | Quintus |
| Sever- | 严肃 | Severus |
| Victor- | 胜利 | Victor、Victoria |
| Claud- | 跛足的 | Claudius |
| Maxim- | 至大 | Maximus |
| Anton- | 无价 | Antonius |
| Flor- | 花/繁荣 | Florus、Florens |
| Soll- | 太阳 | Sol、Soliustia |
| Iust- | 正义 | Iustitia、Iustus |
| Aether- | 以太/天穹 | Aetherius |

**常见族名（星语姓氏）**：

| 族名 | 含义方向 |
| ------ | ------ |
| Metallus | 金属/矿石——梅特卢姆家族的古姓 |
| Draconis | 龙的——与德拉科尼斯地名同源 |
| Aquilius | 鹰的——可能关联阿奎莱亚 |
| Regalis | 王权的——可能关联雷加利亚 |

### 学术/宗教术语构成

星语在帝国学院和教会中用于创造学术与宗教术语。命名模式：

- 取拉丁/希腊混合词根
- 中性名词多用 -um、-ium
- 抽象概念多用 -ia、-tas

例：

- **Soliustia**：sol（太阳）+ iustitia（正义）→ 圣阳
- **Episteme**：希腊语"知识/理解"→ 真知
- **Doxa**：希腊语"意见/信念"→ 共识
- **Aether**：希腊语"上层空气/天穹"→ 以太

# 古英语系

适用地区：阿戈隆德等旧时代派系（与帝国时期的英语做区分）

## 构词规则

### 人名构成

古英语人名同样由词根复合而成，与日耳曼语系同源但更偏盎格鲁-撒克逊风格。相比现代英语名字更粗粝、更古朴。

| 常见词根 | 含义 |
| ------ | ------ |
| Aelf-/Elf- | 精灵/超自然存在 |
| Aethel-/Ethel- | 高贵 |
| Ead-/Ed- | 繁荣/财富 |
| Gar- | 矛 |
| God-/Goth- | 神/善 |
| Here-/Har- | 军队 |
| Wig- | 战斗 |
| Wulf- | 狼 |
| -ric | 统治者 |
| -red | 谋士 |
| -win | 朋友 |
| -mund | 守护者 |
| -stan | 石头 |
| -mere | 湖 |

### 姓氏构成

- **地理特征**：Cragmere（峭壁+湖）、Ashford（灰烬+渡口）
- **职业**：未使用固定姓氏制度时，可以以领地或出身地命名

### 地名构成

- **-ford**：渡口
- **-bury/-burgh**：要塞
- **-ley/-leigh**：林间空地
- **-mere**：湖泊
- **-cester/-chester**：营地（源自拉丁语castra）
- **-ham**：家园/村庄
- **-ton**：围起来的定居点

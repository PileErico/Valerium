# 日耳曼语系

适用地区：德拉科尼斯及帝国核心

## 构词规则

### 人名构成

日耳曼人名通常由两个词根复合而成，每个词根都有独立含义：

| 词根 | 含义 | 举例 |
| ------ | ------ | ------ |
| Adel- | 高贵 | Adelheid（高贵的本性） |
| Bald-/Bold- | 勇敢 | Baldric（勇敢的统治者） |
| Brun- | 铠甲/棕色 | Brunhild（铠甲般的战斗少女） |
| Diet-/Theod- | 人民 | Dietrich（人民的统治者） |
| Fried-/Fred- | 和平 | Friedrich（和平的统治者） |
| Ger- | 矛 | Gerhard（强壮的矛） |
| Hart-/Hard- | 坚强/坚硬 | Hartmut（坚强的精神） |
| Hild- | 战斗 | Hildegard（战斗的守护者） |
| Sieg- | 胜利 | Siegmund（胜利的守护者） |
| Wald-/Walt- | 统治 | Waltraud（统治力量） |
| Wolf- | 狼 | Wolfgang（前进的狼） |
| -ric/-rich | 统治者/权力 | Alaric、Ulrich |
| -mund | 守护 | Siegmund、Rosamund |
| -mar | 名望 | Volkmar、Wilmar |
| -mut | 精神/勇气 | Hartmut |
| -gard | 守护/围栏 | Hildegard |

起名时可自由组合前后词根，例如：Sieg（胜利）+ gard（守护）= Sieggard（胜利的守护者）。

### 姓氏构成

日耳曼姓氏通常来源于以下几类：

- **职业**：Kaufmann（商人）、Zimmermann（木匠）、Jäger（猎人）、Bauer（农夫）
- **地理特征**：Eisenberg（铁山）、Steinhart（坚硬的石头）
- **外貌/特征**：Schwarz（黑色）、Weiss（白色）、Brandt（火焰）
- **动物**：Adler（鹰）、Falkner（猎鹰人）
- **身份/职务**：Richter（法官）、Vogt（管理者）

常见后缀：-mann（人）、-berg（山）、-stein（石）、-burg（堡）、-wald（森林）。

### 地名构成

- **-burg/-berg**：堡/山。Ashburg（灰烬堡）
- **-heim**：家园。Fischheim（费什海姆）
- **-land/-wald**：地/森林
- **-stein**：石

# 煌武

「煌照天下，武定四方」——位于瓦勒瑞姆大陆最东端的大国，拥有毫不逊色于帝国的辽阔领土和独特文化。

---

## 地理

### 大陆位置

煌武位于瓦勒瑞姆大陆最东端。大陆整体呈**"凹"字形、略斜**——凹口在中部偏北，为卡·文图斯的海岸线；西竖为帝国及北境；东竖为煌武，整体比帝国略偏南。

大陆已知势力共七股：阿尔·姆哈、帝国、北境、卡·文图斯、煌武，外加两个尚未设定的派系（北方塞外派系、天山西侧高原派系）。

### 地势

地势西南高、正北高，中部至东部低平。西缘与西南缘为**天山山脉**，构成天然国界；正北有**北方山脉**沿朔州一线分布，使北境地势抬高。天山北端逐渐走低，至西北角地势平缓、敞开一个口子，与卡·文图斯的草原连片相通——此处无山封堵，是煌武对外通道，亦是西北软肋。

海岸仅东、东南面（东海）；东岸自北至南依次为浦州、陉州、沧州。北、西、西南均为陆界。

### 周边势力

- **西北**（垠州外）：卡·文图斯的草原——草原主体及其北面海岸线均归卡·文图斯，煌武不控该海岸
- **西/西南**（隔天山）：天山西侧高原派系（未设定），其西邻阿尔·姆哈、北邻卡·文图斯、东抵天山
- **正北**（朔州/长垣之外）：北方塞外派系（未设定），同时与卡·文图斯和煌武接壤
- **东/东南**：东海

### 山脉

- **天山山脉**：自岚州西缘向南延展至嵚州，沿西缘与西南缘分布。钧天九重圜丘位于嵚州境内的天山段。山脉北端约西中以北逐渐走低，过渡为垠州草原
- **北方山脉**：沿正北朔州一线分布。长垣卡在山脉隘口、补其缺口

### 水系

煌武境内有两条主要河流，均为西东走向。

**沧河**位于南部。性质近长江——辽阔的集中主干，因西、西南地势高而流向集中。发源于西南嵚州/天山，向东南流经沧州，于沧阳入海。陉州东山束其末端北流，逼其保持集中主干。

**泱水**位于中部。性质近淮河——在中央平地上散开成网，多汊、连湖，不收束。因北方山脉抬高，支流整体偏向南部发育。流路自西/西南来，经泱州散成网、衍州汇集、浦州入海（散→汇→泄）。

**天镜湖**位于泱州偏西。原为小湖，开掘永安渠后**永安西渠引水入天镜用于内陆蓄水**，遂涨成大湖。系人工蓄水湖，非天然汇成。

**永安渠**为人工运河，煌武三大奇观之一。分流口在国都天枢。分两支：
- **永安西渠**：自天枢往西南，入天镜湖（蓄水）
- **永安东渠**：自天枢往东南，抵沧阳，汇入沧河

永安渠的核心目的是连通京畿与南方沧阳的水运动脉，兼具双重水利作用：为偏旱的西南调水；分流泱水在衍州一带的过量水、蓄入天镜，缓解衍州的水患。

### 九州

煌武全境划为九州，命名体系取单字+州，禹贡式的朴素质感，以煌武自身地理立名。

**泱水以北（边防低开发带）：**

| 州 | 位置 | 特征 |
|----|------|------|
| **朔州** | 横贯北境，煌武最北 | 北方山脉+长垣；地势高、人烟稀，以驻军边防为主 |
| **垠州** | 西北，泱水以北 | 半干旱内陆草原，接卡·文图斯；温差大、多风 |

**泱水以南（核心开发区）：**

| 州 | 位置 | 特征 |
|----|------|------|
| **岚州** | 西，泱水以南 | 西依天山；西部山前丘陵+东部沼泽，东邻天镜；湿润多雾 |
| **嵚州** | 西南 | 天山南段+沧河源头；钧天九重圜丘所在；高山高寒 |
| **泱州** | 中央偏南 | **京畿，天枢国都所在**；辖永安渠分流口、一段泱水、天镜湖；四季分明、湿润温和、最宜耕 |
| **衍州** | 泱州与浦州之间 | 泱水诸汊汇集之地；原水患频仍，永安渠分流+天镜蓄水后方宜居 |
| **浦州** | 东北 | 泱水下游入海口；东北海岸海防/海口 |
| **陉州** | 泱衍浦三州与沧州之间 | 两山夹一隘，南北咽喉；西山分隔泱沧两大水系，东山束沧河北流；仅一小段海岸 |
| **沧州** | 东南 | 沧河下游出海口；核心城**沧阳**，西临永安东渠、南临沧河；沿海湿润温和多雨。正式/列举作"沧州"，全名可作"临沧" |

### 奇观

| 奇观 | 位置 | 简述 |
|------|------|------|
| **永安渠** | 中部，连通泱沧 | 人工运河，煌武南北大动脉兼水利枢纽 |
| **长垣** | 北部边境（朔州） | 横贯北境的巨型防御工事，卡北方山脉隘口 |
| **钧天九重圜丘** | 嵚州天山 | 煌主祭天之所，煌武最高等级礼仪建筑 |

## 政治制度

### 权力更替

与帝国依靠血脉传承不同，煌武沿袭类似禅让的制度——统治者由民众认可的德才兼备之人出任。煌主名义上受命于天，但受制于道。昏庸失德之君会被认为"失道寡德"，继而被推翻。

煌主有多种称谓——"天命煌主""君上"。严格而言煌主只是被民众选出来照顾天下苍生的人，但任何一位合格的煌主，在煌武的土地上都拥有一呼百应的号召力。

> 【待修订】以下官吏与组织架构（一相三部六院、监察司、谏官等）为旧版设定，偏通用的中华帝制范式，尚未把煌武的两个独特性吃进制度——道作为真实因果律，以及合法性源于"被共识登记的认可"。重修方向：凡涉及信息、名声、真相的机构（如谏官"不得报假"那条），本质都是这个文明的共识维护机器，应据此重新立形。下列内容暂存待改。

### 一相三部六院

煌武中央采用金字塔形权力架构，集权度远超帝国。煌主为最高决策与统筹者。

**一相**——相国：煌主首席辅政大臣，统筹协调三部六院事务。对煌主负责，接受监察司监督。

**三部**：

| 部门 | 职能 |
|------|------|
| 军部 | 军政事务、军事部署、战时指挥辅助 |
| 政部 | 国家政策制定与施行，收集地方反馈 |
| 吏部 | 人事任免、考核与户籍管理 |

**六院**：

| 院 | 职能 |
|------|------|
| 法院 | 司法审判、律法修订 |
| 工院 | 工匠、军械、交通、水利、工程建设 |
| 礼院 | 礼仪、祭祀、外交典礼 |
| 学院 | 教育、学术研究、人才培养 |
| 廩院 | 粮仓、税收、漕运与储备 |
| 商院 | 商业、关市、贸易与物资流通 |

### 双重监督

**监察司**——自上而下。直接听命于煌主，成员由三部六院各派代表组成。有权否决政令，监督三部六院与地方官员，处理重大贪腐与失职案件。理论上连煌主也在其监察范围内。监察司本身也接受谏官监督。

**谏官制度**——自下而上。独立于行政与军权，唯一职能是进谏、上达民意。全民选举产生：基层谏官由民众选出，层层推选至可面见煌主。出身不限——乞丐、农夫、商人皆可参选。任期制，可连任。享有最高等级的人身保护，唯一禁令是不得上报虚假信息。

### 制衡模式

- 煌主依靠监察司和谏官制约士族
- 士族通过在三部六院中的席位保持影响
- 谏官选举制让底层声音直达煌主，避免民意被士族与官僚阶层阻隔

## 文化

- **尚武**：习武是日常美德，但并不意味着武人垄断政权
- **崇直尚言**：敢言、敢谏是社会认可的价值观，不分出身
- **实用主义**：制度和职位不为装饰，为解决问题、维持秩序

## 信仰

煌武不推崇人格化的神明信仰——它的哲学里根本没有"神明"这个位置，一切自有**道**的安排。道不是外在之神，而是煌武人世代共持的、对世界终极规律的认知；正因世代共持，道在煌武境内成为真实运作的因果律。循道者为有德、失德则寡助；煌主"受命于天"，在天山圜丘祭天，而天不是神，是这份共同认知所指向的终极秩序。（民间另有拜保家仙、灶王爷一类的零星风俗，是民俗边角，不是主干。）

道作为道家哲学的完整展开——认知如何判人、政治场上道即民众认可、"人/民"的边界、及其在历史尺度上的运作——详见 信仰／道。

## 庚辰

煌武的图腾是**应龙**庚辰——一头高知性的强大巨龙，从煌武人世代供奉中获得守护者、圣兽的位格。它曾袭击煌武，却反而促成了煌武空前的团结：煌武人群起相抗，终将它逼上谈判桌、立约为守护者。能齐心迫使一头巨龙俯首，本身就是煌武天命最好的背书。

后庚辰遭政变者勾结外部势力袭击而长眠，煌主再不能呼唤它——煌武一向倚仗的那支外部抑制力就此撤去，本就在积累的统治弊端、王朝末期的分崩（道在历史尺度上的必然）再无人摁住。庚辰长眠是乱的扳机，而非乱的根。

（庚辰作为独立个体的详档见 角色／庚辰。）

# 煌武

「煌照天下，武定四方」——位于瓦勒瑞姆大陆最东端的大国，拥有毫不逊色于帝国的辽阔领土和独特文化。

---

## 地理

### 地形

煌武地势西高东低。西部为戈壁与高原，与阿尔·姆哈东部边境之间隔着广阔干旱地带。西北接卡·文图斯，多为草原与丘陵的过渡区域。东部为冲积平原，是人口与农业的核心地带。

### 水系

煌武境内有两条主要河流，均为西东走向，自高原发源东流入海。

**沧河**位于南部。上游流经山地与平原过渡带，下游至出海口一带为临沧州，核心城市沧阳坐落于沧河北岸。沧河是煌武南部农业与漕运的命脉。

**洛水**位于中部，汇入内陆湖**天镜**。天镜为煌武第一大湖，洛水流域是煌武政治与文化的心脏地带。

**永安渠**是连接洛水与沧河的人工运河，煌武三大奇观之一。南北贯通后，洛沧两大流域联为一体，漕运、商贸、兵力调配皆仰赖此渠。

### 州郡

煌武全境划为九州。目前已有名称者：

- **临沧州**：沧河下游至出海口。核心城沧阳
- **天枢**：七星之首，煌武国都。位于洛水—永安渠交汇处附近

九州其余各州、七星余下六城待定。

### 奇观

| 奇观 | 位置 | 简述 |
|------|------|------|
| **永安渠** | 中部，连通洛沧 | 人工运河，煌武南北大动脉 |
| **长垣** | 北部边境 | 横贯北境的巨型防御工事 |
| **钧天九重圜丘** | 西部天山 | 煌主祭天之所，煌武最高等级的礼仪建筑 |

## 政治制度

### 权力更替

与帝国依靠血脉传承不同，煌武沿袭类似禅让的制度——统治者由民众认可的德才兼备之人出任。煌主名义上受命于天，但受制于道。昏庸失德之君会被认为"失道寡德"，继而被推翻。

煌主有多种称谓——"天命煌主""君上"。严格而言煌主只是被民众选出来照顾天下苍生的人，但任何一位合格的煌主，在煌武的土地上都拥有一呼百应的号召力。

### 一相三部六院

煌武中央采用金字塔形权力架构，集权度远超帝国。煌主为最高决策与统筹者。

**一相**——相国：煌主首席辅政大臣，统筹协调三部六院事务。对煌主负责，接受监察司监督。

**三部**：

| 部门 | 职能 |
|------|------|
| 军部 | 军政事务、军事部署、战时指挥辅助 |
| 政部 | 国家政策制定与施行，收集地方反馈 |
| 吏部 | 人事任免、考核与户籍管理 |

**六院**：

| 院 | 职能 |
|------|------|
| 法院 | 司法审判、律法修订 |
| 工院 | 工匠、军械、交通、水利、工程建设 |
| 礼院 | 礼仪、祭祀、外交典礼 |
| 学院 | 教育、学术研究、人才培养 |
| 廩院 | 粮仓、税收、漕运与储备 |
| 商院 | 商业、关市、贸易与物资流通 |

### 双重监督

**监察司**——自上而下。直接听命于煌主，成员由三部六院各派代表组成。有权否决政令，监督三部六院与地方官员，处理重大贪腐与失职案件。理论上连煌主也在其监察范围内。监察司本身也接受谏官监督。

**谏官制度**——自下而上。独立于行政与军权，唯一职能是进谏、上达民意。全民选举产生：基层谏官由民众选出，层层推选至可面见煌主。出身不限——乞丐、农夫、商人皆可参选。任期制，可连任。享有最高等级的人身保护，唯一禁令是不得上报虚假信息。

### 制衡模式

- 煌主依靠监察司和谏官制约士族
- 士族通过在三部六院中的席位保持影响
- 谏官选举制让底层声音直达煌主，避免民意被士族与官僚阶层阻隔

## 文化

- **尚武**：习武是日常美德，但并不意味着武人垄断政权
- **崇直尚言**：敢言、敢谏是社会认可的价值观，不分出身
- **实用主义**：制度和职位不为装饰，为解决问题、维持秩序

## 信仰

煌武不推崇人格化的神明信仰，坚信一切自有**道**的安排。道并非外在之神，而是人们对世界运行规律的认知——如同自然规律或真理。煌武人世代持有对道的认知，使道真正成为了在煌武境内真实运作的因果律。

道以两种方式影响煌武之人：

- **因果**：善恶有报、事出有因——看似偶然背后皆为必然。西方向神立誓，东方向天。煌武人极为忌讳以类似预言的方式窥探因果
- **德**：道之得、道之用。言行循道者为有德之人。顺应道者可展现超凡力量，但德不强制约束——失德则寡助

道与天的关系：道是规律，天是规律的最高象征。煌主"受命于天"，在天山圜丘祭天——但天本身不是神明，是煌武人对终极秩序的共同认知所指向的方向。

## 庚辰

煌武的图腾是**应龙**。按帝国学者的定义，应龙属于巨龙种——四足双翼，拥有高度智力和强大力量的个体特化龙类。

应龙名为**庚辰**。数千年来，它一直以圣兽身份受到煌武人民的崇拜和敬畏，是煌武政权的支柱与象征。后遭政变者勾结外部势力袭击而陷入沉睡，煌武因此陷入短期动荡。

# 力量与规则

这个世界一切超出寻常物质规律的现象——施法、奇迹、生物本能、化形——都建立在两类不同性质的根基之上。

**[真知（Episteme）](真知（Episteme）.md)** 是一种面对世界的认识立场——指认那些"本然如此"的客观规律。物质层的所有规律都被真知所指认：以太作为底层物理场的性质、生物在以太环境中的演化、质能转换……以及共识作为机制本身的存在。真知本身不是一条规则，它是一个总集——一切客观规律的总称。

**[共识（Doxa）](共识（Doxa）.md)** 是真知所指认的诸多规律中的一条特殊规律：知性生物的认知拥有向下覆写物质层的力量。它的特殊在于它持续地为真知扩增内容——被共识塑造而立起来的事物（如煌武境内的"道"、圣阳教的索利斯），一旦在物质层运作，就成了真知所包含的新规律。

严格地说，共识是真知的一部分，不与真知并列。但作为这个世界两类截然不同的现象根源——以太路径上的现象（建立在真知所指认的固有物理之上）和共识路径上的现象（建立在认知覆写物质这条特殊规律之上）——真知与共识在分类层面被并列处理，构成本文档下的两个大节。

各文明对真知与共识的认识姿态不同。帝国把两者分清作平行的两条路径——以太利用作为一类范式、奇迹与信仰作为另一类；[煌武](../06-派系/煌武.md) 则不作此分，让两条在文明的认知和实践中浑然一体——煌武的"道"既是真知层意义上的规律，也是共识塑造的具体内容，二者无法被切割。

## 以 [以太（Aether）](以太（Aether）.md) 为基础

以太是真知所指认的底层物理场，承载魔力。它可被知性生物的意志强行抽取能量做工——这是一切以以太为基础的力量路径成立的根基。

各文明以不同方式利用以太，演化出几条不同的范式。

### 帝国的施法范式

帝国把以太利用按"作用对象在外还是在自身"分为两支——

#### [理术（Techne）](理术（Techne）.md)

对外应用——以太能量作用于外界（物质、以太自身、力场）。

- 元素（Element）
- 塑能（Etherics）
- 场域（Force）

#### [体术（Askesis）](体术（Askesis）.md)

对内训练——以太能量作用于施法者自身肉体。

- 强化
- 战技

### 煌武的修炼范式

#### [功法](功法.md)

煌武不作"对内 / 对外"的分别——功法是内外兼修的整体。其下分三种修炼路径，彼此不兼容——

- **气修**：吐纳调息、引导内气循环精进自身
- **体修**：反复锤炼肉体，含 Doxa 加成的"刀枪不入"
- **丹修**：道家内丹术，借假修真，传统理想为得道飞升

煌武也不区分真知与共识——功法建立在以太（气）的运行之上，但其前提（丹田、经脉、穴位等）是煌武共识所塑造的真知层规律。两条路径在功法中浑然一体，是煌武体系的根本特色。

### 生物的先天能力

#### [特化（Physis）](特化（Physis）.md)

生物作为其所是所本然携带的、与以太相关的生物学特征——物种身份本身带来的"硬件基础"，无需后天学习。常见来源是自然演化（飞龙的以太驱动肌肉、各种以太怪物的特殊器官等），但工程造物（[不朽者](../02-种族/不朽者.md)）、基因植入（[精灵](../02-种族/精灵.md)）、仪式整合（龙裔）等已成为本然属性的特征同样归入此类。

与上述各文明主动利用以太的范式不同——特化是被以太塑造的本性，而非主动施展的技艺。

## 以 [共识（Doxa）](共识（Doxa）.md) 为基础

共识是真知所指认的一条特殊规律——知性生物的认知能向下覆写物质层。它的运作以信念（Pistis）作为输入、映像（Eikasia）作为输出，在物质层兑现各种现象。

共识路径下的现象类别繁多，且各类之间常有交叉。粗分如下——

### 信仰与奇迹

特定信念被广泛持有时，可催生神明（无载体的概念性实在如索利斯，或附着于真实载体的神格化如库库尔坎）。被信徒承认践行了道途的修士，可在神明权能下施展奇迹——治愈、赐福、占卜等。详见 [信仰](../05-信仰/README.md) 相关文档。

### [禁制（Geis）](禁制（Geis）.md)

共识的等价交换应用——通过代价与效果配对兑现映像。立誓者以付出代价或遵守限制换取超出寻常的力量；诅咒、祝福等可视作其代价 / 效果归属关系不同的具体延伸。代价的"重度"不可量化，由付出者主观对失去的在乎程度决定。北方蛮族文化中尤为典型。

### 共识所塑造的实在

被持续认知的概念，可在物质层立起为真实运作的规律或存在——煌武境内的"道"作为因果律、各种契约神兽（如庚辰）、被信仰兑现的圣物（如淬阳武器）、被指认为森林意志的塞纳诺斯等等。详见 [共识（Doxa）](共识（Doxa）.md) 及相关条目文档。

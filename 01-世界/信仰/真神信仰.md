# 真神信仰
 
阿尔·姆哈文明的本土宗教。标准一神教，以真神（创世神）为终极至高存在，下设三神（主神）与次级神构成完整神明体系。原本是阿尔·姆哈作为统一王国的国教，蛇人事件后随遗民西迁至雷加利亚境内继续传承至今。是 [圣阳信仰](圣阳信仰.md) 的远源。
 
---
 
## 信仰结构
 
### 三层神明体系
 
| 层级 | 形象 | 例子 |
|---|---|---|
| **真神 / 创世神** | 不可被认知 | 唯一 |
| **三神（主神）** | **人形象** | 太阳、绿洲、冥河 |
| **次级神** | **兽首形象**（类似埃及神祇） | 腐败之神、暴力之神等 |
 
形象本身编码了等级——
 
- **真神**：连形象的描述都不存在。绝对至高，凡人不被允许接近这个层级
- **三神**：完整的人形——和凡人形态相似但崇高。这是权威"接近凡人但仍在凡人之上"的可见层
- **次级神**：兽首人身——保留人形躯体但顶着兽首。"完整的人形"是高级别的标志，"带兽首"立刻显示出他们是低于三神的存在
凡人看到任何一尊神像，从形象就能立刻判断其在体系里的位置。
 
### 真神
 
真神是创世神，是绝对至高的主宰。
 
**所有人都知道真神的存在，但所有人对真神的认知都停在"知道他存在"这一层**——
 
- 不知道真神具体是什么样
- 不知道他的名讳
- 不知道关于他的具体内容
- 也**不应该想知道**
这并非单一的禁忌，而是阿尔·姆哈整套**"权威不可侵犯"文化**在最高层的体现。在阿尔·姆哈的世界观里，下级对上级做任何**可能挑战、削弱、抹平那份权威**的事都是冒犯——打探、质疑、衡量、判断、僭越行事，全部属于这一类。
 
世俗层面：下层不能随便了解、质疑苏丹与贵族的私事。
宗教层面：凡人不能去认识、衡量、思考真神。
 
这是同一套逻辑——真神是这套逻辑的终极顶点。"认知"真神本身就在把他拉到可被你审视的位置，这就削弱了他作为绝对至高存在的那份权威。**真神之所以是真神，就在于他绝对在你之上、不可被拉下来审视**。
 
凡人对真神的存在认知是通过三神间接得到的——既然眼前的三神都有如此伟力（奇迹），那缔造了他们的又该是何等伟大的存在？这是所有阿尔·姆哈人都能自然推出的结论：真神必然存在、必然远高于三神。但仅止于此。
 
**真神是一个"人尽皆知却又微妙地知道得不太多"的伟大存在**。

### 三神
 
三神是真神的造物，是凡人能直接接触、祭祀、获得奇迹的层级。
 
三神各自独立，有各自的神格、性格、职能、奇迹谱系——不是真神的"三种侧面"或"三相"，是三个独立的神明。
 
#### 太阳
 
代表**权威、力量、统治**。
 
对应阿尔·姆哈本身的严苛阶级和上位者神赐的权柄。是阿尔·姆哈最显赫、最直接被崇拜的神——苏丹与上位贵族的统治正当性在很大程度上锚定在这一神格上。
 
太阳神的奇迹谱系包括如**"律令"**——可以强制命令他人服从。这条权能直接对应统一王国上位者的实际需求。
 
#### 绿洲
 
代表**生命、源头、家乡**。
 
在沙漠环境中，绿洲是生存的根本——这条意象天然连接着生命本身、族群的起点、归属之地。
 
#### 冥河
 
代表**死亡、夜晚、永恒的宁静**。
 
与绿洲形成生死对照——绿洲是生的开始，冥河是死的归宿。
 
### 次级神
 
三神之下还有数位次级神，覆盖更具体、更日常的领域——比如**腐败之神**、**暴力之神**等。
 
兽首人身的形象本身就标识了他们的等级——一眼可辨。但等级低不等于无足轻重——凡人日常面对的不只是宏大的力量 / 生命 / 死亡，也面对具体的腐败、具体的暴力、具体的各种处境。这些**具体处境对应的奇迹**就由次级神负责。
 
【待补充：次级神的具体清单与各自职能】
 
### 凡人的祭祀
 
凡人的祭祀活动**面向三神和次级神**——不直接面向真神（真神不可被认知，无法被祭祀）。
 
根据具体需要选择对应的神：祈求统治力量找太阳，祈求生命与归属找绿洲，安抚亡者面对死亡找冥河，对应具体处境的找相应的次级神。
 
奇迹力量从这些"低于真神一级"的神明处获得——这是凡人能接触到的全部神圣力量。

## 历史
 
### 兴起
 
阿尔·姆哈这片土地的历史远早于人类——数千万至上亿年前曾是雨林，居住着爬行种文明（详见 [爬行种的神](爬行种的神.md)）。爬行种为躲避寒潮在地下冬眠，地表经过漫长地质变迁演化为沙漠。
 
后来兴起的人类文明在这片沙漠之上立国——这就是阿尔·姆哈。真神 + 三神 + 次级神的信仰体系随这个文明的发展而成熟，与阿尔·姆哈的社会结构（严苛阶级、世袭苏丹制、权威不可侵犯的层级文化）深度匹配、互相强化。
 
数百年间阿尔·姆哈作为统一王国发展出极其发达的宗教体系——完善的神学、成熟的仪式、系统化的认知框架、真实可调用的奇迹。在当时的西方还是一群蛮族割据的时期，阿尔·姆哈已经是世袭苏丹统治下的统一王国。
 
### 蛇人苏醒与诱骗
 
数千万年前在地下冬眠的爬行种里，**只有蛇人苏醒**。他们出土后发现地表已被人类占据。
 
蛇人没有正面冲突，而是通过深度欺骗操纵阿尔·姆哈苏丹——让苏丹率众改信"爬行种的神"。但这套"爬行种的神"完全由蛇人描绘——原本更复杂的爬行种神明体系**已经被坍缩为单一的蛇神**。
 
蛇人许诺以永生。但实际结果是阿尔·姆哈所有子民最终被改造成蜥蜴人，沦为蛇人的奴隶。真神 + 三神信仰也随之被替换为蛇神信仰。
 
详细的爬行种神明体系演变见 [爬行种的神](爬行种的神.md)。
 
### 哈立德的抗拒与西迁
 
[哈立德·阿尔·扎希尔](../06-派系/雷加利亚.md)——一位埃米尔——**从一开始就抗拒蛇人**，没有被蛇人波及。他遭到蛇人讨伐，无法正面对抗，只能边对抗边撤离，最终率领自己的子民西迁。
 
哈立德拒绝苏丹改信蛇神的命令——正是因为他**更看重自己的意志和阿尔·姆哈本来的信仰**。他是虔诚的真神信徒，这一点毫无疑问。亲历"权威被滥用到几乎吞没整个文明"之后，他对所谓权威的一意孤行多少有了迷茫——觉得它不再那么绝对——但这并不是不再信仰，更不是动摇了对真神的虔诚。
 
作为遗民领袖，哈立德也不会主动表露这种迷茫——异乡求存的群体需要明确的领袖锚点，主动外露内心的不确定等于自我抹除应有的权威。何况这些能跟随他活下来的人，正是对原生信仰怀有强烈情怀的人——直到帝国时期他们都没有割舍自己的原生信仰。哈立德对此心知肚明。
 
阿尔·姆哈遗民作为一个群体把整套真神信仰带到了西方——信仰是他们作为群体共同持有、共同传承的，不属于某个个人。

## 现状
 
### 阿尔·姆哈遗民聚居在哈立德辖区
 
哈立德加入九骑士、为难民争取一席之地——九骑士各自有自己的辖区，**哈立德的辖区是阿尔·姆哈人的主要居住地，并一直延续到帝国时期**。
 
聚居的原因不只是哈立德的领导——他们的习俗、文化、信仰和当地人都不一样，**没有那么融洽**。聚在一起是顺理成章的选择。
 
作为雷加利亚的**地方民族**，他们有自己的辖区，并有**一定程度的自治**。
 
### 信仰传承
 
聚居密度高 + 几代人持续守护 → **信仰传承得比较完整**。
 
按 [共识（Doxa）](../04-力量与规则/共识（Doxa）.md) 机制，真神信仰仍然是一个活的信仰：
 
- **规模**：被西迁和漫长岁月削减，远小于阿尔·姆哈鼎盛时期
- **同步度**：聚居 + 文化高度内聚 → 极高
- **持续度**：几代人不间断 → 极高
- **精确度**：神职传承完整 → 较高
同步度 + 持续度 + 精确度补偿了规模的不足 → **依然能调用奇迹**，包括三神和次级神各自的奇迹谱系。
 
### 语言
 
主体继续使用**阿尔·姆哈语**，也会使用**少量的星语**（详见 GitHub 仓库的星语 / 星民条目）。
 
### 外界态度
 
外界对阿尔·姆哈遗民的认知是分裂的：
 
- 有人认为他们是**乡巴佬**——经济、文化、生活方式不被主流看得起
- 有人认为他们是**可怜的流浪民族**——同情的视角
- 很多人甚至**不清楚阿尔·姆哈是什么东西**——遗忘 / 无知的视角
大部分圣阳信徒并不知道圣阳的源头追溯到阿尔·姆哈太阳神——这种主流的"不知道"在阿尔·姆哈人遭遇外界态度时也呈现出来。
 
### 民族目的
 
阿尔·姆哈人作为民族高度专注于两件事：
 
- **持续专注于复国**——夺回故土、重建王国
- **将蜥蜴人变回人类**——让被蛇人改造的同胞从痛苦中解脱
这两件事都是几代人持续的事业，让他们作为一个民族保持高度的目的性和内聚性——这反过来强化了信仰的持续度。
 
### 圣战远征中的合流
 
[圣阳教会](圣阳信仰.md) 多次发起圣战远征夺回圣城。每次远征**阿尔·姆哈人都很积极参与**——尽管彼此的信仰早就分化了。
 
两边的目的不一样但**顺路**：
 
- 圣阳教会：宗教目的，夺回圣城乌比索·奥利图尔
- 阿尔·姆哈人：复国、对蛇人复仇、收复故土
目标方向都指向同一个东方——所以圣战远征和复国战争实际上是同一支军队、不同动机的合流。

## 奇迹
 
### 机制层面
 
真神信仰的奇迹和圣阳奇迹在**机制层面是相同的**——都是 [共识（Doxa）](../04-力量与规则/共识（Doxa）.md) 这一普世规律的应用。
 
不同的只是**对应的概念和效果**——奇迹兑现的具体内容由信仰的内容决定。
 
### 三神奇迹谱系
 
三神各自有独立的奇迹谱系，对应各自的神格内涵：
 
- **太阳神的奇迹**：力量、统治、权威相关的兑现。例：**律令**——可以强制命令他人服从

### 与圣阳奇迹的关系
 
**真神信仰的奇迹和圣阳奇迹是不重叠的两套谱系**。
 
太阳神的"律令"在圣阳里完全没有继承——圣阳奇迹围绕力量（捍卫弱者）/ 公正（伸张正义）/ 疗愈 / 驱散黑暗 / 包容 / 赐福，没有"强制命令他人服从"这种形态。
 
次级神的"腐败"、"暴力"等奇迹更是完全不在圣阳的谱系里——西部接受方渴望的是被庇护、被救助、向善的内容，没有需要负面奇迹的土壤。
 
两套奇迹在帝国境内并行不悖——各自由各自的信念体系支撑兑现，互不冲突。阿尔·姆哈遗民调用三神 + 次级神奇迹，圣阳信徒调用圣阳奇迹。

## 与圣阳的关系
 
[圣阳](圣阳信仰.md) 和真神信仰是**两个独立的信仰、两个独立的宗教**——没有承接关系。
 
"圣阳源于三神信仰"指的是**逻辑因果**——没有真神信仰先到了西部，圣阳就不会以现在的形态出现。但这只是逻辑层面的依赖，不是宗教层面的承接。
 
圣阳沿用真神信仰，走的是一条偷懒的套用之路：把现成的外在内容（仪式、传说、圣城等）拿过来接着用，而非另起炉灶。但两者作为宗教始终是独立的，套用是另一回事。
 
归结起来，圣阳之于真神信仰，是**独立 + 偷懒套用 + 套用之后独立演化**的关系。
 
**套用的实际机制**：
 
- 阿尔·姆哈传教者讲过的内容（教义骨架、仪式、圣城传说等）被听过的人记住一部分
- 后来这些人中的一些改信了圣阳——**他们做事的方式不会被清空**，继续做这些礼拜动作、传讲这些传说，只是心里指向的对象换成了圣阳
- 新加入的信徒看到他们这么做、跟着学；圣城传说在信徒之间互相传话、添油加醋，慢慢成为圣阳信徒群体里大家都知道的事
- 后续如果有不合圣阳味道的细节，再逐渐改造
- 但圣阳作为信仰的核心（信什么、奇迹谱系、信徒群体、神格存在与否）和真神信仰**完全不同**——这些是圣阳独立形成的部分
阿尔·姆哈遗民这一边则**始终守着完整的真神 + 三神信仰**，没有把它演化成圣阳。两条线分别发展，最终成为两个独立的信仰体系

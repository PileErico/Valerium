# 力量与规则

本世界存在两条底层规则——

- **[真知（Episteme）](真知（Episteme）.md)**：物质层客观规律，独立于任何观察者而存在
- **[共识（Doxa）](共识（Doxa）.md)**：知性生物的认知向下覆写物质层的力量

一切超出寻常物质规律的现象——施法、奇迹、生物本能、化形——皆建立在这两条规则之上，并衍生出两条本质截然不同的路径。

## 以 [以太（Aether）](以太（Aether）.md) 为基础

由真知所揭示的物质粒子"以太"承载魔力，各文明以不同方式发展出对其的认知与利用。

### [理术（Techne）](理术（Techne）.md)

帝国"对外应用"范式。

- 元素（Element）
- 场域（Force）
- 塑能（Etherics）

### [体术（Askesis）](体术（Askesis）.md)

帝国"对内训练"范式。

- 强化
- 武技

### [功法](功法.md)

煌武"对内训练"范式。

- 内功
- 外功

### [特化（Physis）](特化（Physis）.md)

生物在以太环境中演化所形成的先天能力，非习得。

## 以 [共识（Doxa）](共识（Doxa）.md) 为基础

与 [以太（Aether）](以太（Aether）.md) 不同，共识（Doxa）本身不是一种存在的实体，它是一种规则，是对特定规律和现象的一种总结。其主要由信念（Pistis）和映象（Eikasia）组成。信念是知性生物的具体认知，而映像则是因为这些认知产生的具体现象。

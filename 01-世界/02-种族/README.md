# 种族

这个世界的智慧生物并不全是自然演化的产物。有些来自漫长的进化，有些则诞生于远古文明的实验与意外。它们之间的差异不仅体现在外貌和能力上，更深层的区别在于构成身体的基础物质——碳基的血肉，或是以太粒子的凝聚。

---

### [人类](人类.md)

适应力极强，生命短暂却善于传承。生理强度不占优势，但凭主观能动性改造环境、积累知识，成为世界上最具影响力的存在。文化习俗和信仰体系因地域而截然不同，由此引发的内战有时甚至比异种族间的冲突更为激烈。

### [不朽者（The Eikon）](不朽者.md)

上个世代人类文明的造物。以以太粒子逐粒复刻人体而造出的永生之躯——身上每一个分子结构都与人类蓝本一致，唯一不同的是全部基本粒子都是以太。上个世代共造出数十个，文明覆灭后大部分在漫长岁月中选择了自我了断。至帝国历仅存不足十个，四处流浪，靠某种执念支撑。

### [精灵](精灵.md)

某位不朽者一时兴起的产物，实为不朽者的劣化版本。血统浓度越高，寿命和魔力越强；稀释到与人类趋近后即为半精灵——因寿命长、耐用而在某些地区沦为奴隶市场的商品。只在大陆西部的大森林及附近出没，没有记录历史的习惯。

### [鬣狗人](鬣狗人.md)

与鬣狗极为相似的异形种族，母系社会，被饥饿感驱使。原本生活在南方，因气候变化北迁后曾以穴兔为猎物，却反被崛起的穴兔驯化奴役。在长期囚禁中被迫产生思考，萨满将所有个体共享的饥饿与仇恨统一为明确的认知，由此引发 Doxa 层面的质变——最终凭借歪曲现实的力量挣脱牢笼，成为穴兔的永恒死敌。

### [鱼人](鱼人.md)

诞生于灵渊的种族，孵化时经历各种突变。近岸的两栖鱼人已与深海同族踏上完全不同的进化道路——尾部退化、长出行走用的腿部，智力稳步提高，连信仰都从灵渊的"生命温床"转向了海洋本身的"自由与财富"。甚至有个体在费什海姆成为正式冒险者。

### [爬行种](爬行种.md)

阿尔·姆哈的远古文明种族，种姓严格：龙人（领袖）、蛇人（神职，智力极高，将信仰当作可操控的能量机制）、蜥蜴人（底层）。现存的蜥蜴人大多源自被蛇人献祭仪式转化的人类，以商人身份出没于大陆各地，家族观念很强。

### [穴兔](兽人.md)

卡·文图斯的主体种族（Leporian）。原型为穴兔，在以太环境下演化出知性与强壮体格。【设定详见卡·文图斯派系文档】

### [矮人](矮人.md) 【待定】

存在于历史记录中，最终形态未确定。

### [龙裔](龙裔.md)

并非独立种族，而是通过汲取古龙之血蜕变的人类。德拉科尼斯猎龙家族的传统是狩猎兽龙、沐浴龙血以淬炼身体，但温德林更进一步——利用古龙贝希摩斯的封存之血完成了重生仪式，以大半参与者的性命为代价，让自己成为第一个体内流淌龙血的龙裔。随后屠尽其他幸存者，确保血脉专属于欧恩斯坦家族。龙裔的诞生成为征服战争的导火索，改变了整个大陆的力量格局。

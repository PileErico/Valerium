# 穴兔（Leporian）

卡·文图斯的主体种族。演化自穴兔，在以太环境中发展出足以支撑原始文明的知性，同时获得了更为强壮的体格。

| 特征 | 说明 |
| ---- | ---- |
| **体型** | 矮小但结实，与哥布林或矮人相当 |
| **繁殖率** | 因体型增大而有所下降，但相对其他知性种族仍然极高 |
| **幼体发育** | 晚熟型，依赖母体长期照护 |
| **性情** | 高度攻击性。频繁的种内冲突并非文化行为，而是进化筛选的结果——温顺个体在淘汰中不具备生存优势 |

---

## 前肢结构与文明瓶颈

穴兔的前肢结构继承自其穴居原型，是制约其文明发展的根本因素。前掌粗壮有力，适于挖掘，但不具备精细操作能力：

- **掌心覆毛，无肉垫**：握持工具时缺乏摩擦力，需依赖外来工匠制作的皮质手套辅助抓握
- **指节数量有限，灵活度低**：可执行握、砸、撕、拉等粗放动作，但无法完成纺织、雕刻、精密锻造等工艺
- **腕部活动度高**：穴兔保留了原型物种灵活的腕关节结构，使其在使用短兵器时拥有极高的转腕自由度

因此，穴兔能够使用武器、开掘地道、驱使战兽，但**无法自行生产**精良的工具与武器装备。这一缺口只能通过对外劫掠来填补——这并非文化选择，而是生理结构决定的必然。

---

## 以太影响

以太对穴兔的作用并不显著。穴兔的知性与体格均为自然演化的产物，以太并未在其中扮演关键角色。

唯一可归因于以太的生理特征是异常的**韧性**——穴兔在遭受常规致命伤后仍可维持行动能力，其存活阈值远高于同体型的其他物种。这一特性使得穴兔之间的淘汰厮杀异常惨烈：双方均难以被迅速杀死，每一场对决都会演变为漫长的消耗战。由此，能够在淘汰中存活的裂耳（The Split）所经受的筛选烈度远超常规，每一个幸存者都是经过极端考验的个体。

穴兔不具备主动运用以太的能力。施法要求高度抽象思维与系统性知识积累，这超出了穴兔的认知范畴。

---

## 武装与战斗

受前肢结构与地下作战环境的双重制约，穴兔的武装和战术高度特化：

- **以短兵器为主**：短剑、匕首、短柄战斧。适配地道中的狭窄空间，无需大幅度挥砍
- **刁钻的近身刺击**：得益于腕部的高活动度，穴兔武士在狭窄通道中的突刺角度极为诡异，近身战能力极强
- **与旷兔协同作战**：旷兔作为战兽而非坐骑参与战斗，兔武士与旷兔并肩冲锋，形态类似人类使用猎犬，但旷兔的体型和战斗力远超后者
- **无法运用复合装备**：重型武器、长枪方阵、精密弩机等需要精加工或高度协同的装备，穴兔既无法制造也难以有效操作

## 旷兔

穴兔的地表远亲。进化时间晚于穴兔，未能发展出独立的社会结构，在穴兔长期有意识的选育下体型逐渐增大。

| 特征 | 说明 |
| ---- | ---- |
| **体格** | 大于穴兔本体，体态接近小型熊类 |
| **感官** | 视觉与听觉敏锐 |
| **四肢** | 腿部较长，擅长奔跑 |

旷兔并非坐骑。穴兔的弓形脊椎结构不适合承载骑乘者，骑乘会干扰旷兔的运动。其主要用途为**突击**与**驼运**：在战斗中作为冲击力量向目标发起冲锋，日常则负责运输武器、食物和装备。

旷兔的培育由底层杂鱼负责，选育标准直接且原始：更快、更大、更强，不涉及复杂的知识传承。

## 等级体系

穴兔社会的等级结构可通过耳朵的物理状态直观辨识，各等级因此获得了对应的称谓。

### 汗（The One）

族群中战斗力最强的个体，唯一的领袖。汗是族群中被允许拥有多名配偶的唯一个体；其配偶被视为私有财产，汗死后由继任者处置。

### 立耳（The Stand）

汗的近卫，由族群中几乎从未在战斗中负伤的顶级个体组成。唯有这些受到信任的个体被允许在汗面前竖起耳朵——这一姿态本身即为地位与战力的象征。

"立耳"这一称谓的含义并非"耳朵完好无损"，而是"**强大到几乎无人能触及其耳朵**"。耳朵上的伤疤越少，意味着该个体越危险。

**权力交接**：在位汗仍具备战斗力时，立耳之间会联合抑制任何挑战行为——维持现有秩序符合立耳集团的共同利益。仅当汗因战死、衰老或伤残而无法维持"最强"这一事实时，立耳之间才会爆发争夺。每次换代是一场短暂而惨烈的内战，但并非常态。

汗不干预战争以外的事务，立耳的交配行为完全自发。

### 裂耳（The Split）

普通武士阶层。"裂耳"指其耳朵在长期战斗中形成的撕裂痕迹。裂耳是末位淘汰制的幸存者——获得这一身份仅证明该个体具备基本的战斗素质。

裂耳对立耳的态度并非纯粹的服从，其中夹杂着本能层面的不满与敌意，但因实力差距而被压抑。这种积蓄的攻击性通常在汗换代时期集中释放。

### 底层（杂鱼）

底层并非固定身份，而是一个**连续滑动的光谱**。根据个体的健康状态和行为能力，光谱上端接近奴隶，从事基础劳作（喂养旷兔、搬运物资、挖掘地道、辅助战斗）；下端接近储备粮。个体因伤病或衰老而向下滑落的过程不可逆。

底层之间不存在团结。靠前的个体会主动攻击更弱的同类以维持自身的利用价值——这并非恶意行为，而是求生本能的直接表达。

---

## 信仰

穴兔不具备任何形式的信仰体系。其全部生存资源被投入于战斗、繁殖和劫掠，没有发展形而上认知的余裕。这与拥有 The Hunger 的[鬣狗人](鬣狗人.md)以及具备多元神明崇拜的人类形成鲜明对比——穴兔是已知知性物种中罕见的纯功利性文明。

---

## 血脉分化

汗作为最强个体独占多配偶特权，其后代在先天素质上具备统计优势。经数十代积累，这种强者优先交配的模式自然产生了事实上的贵族血统——并非有意识的选育计划，而是"最强者交配最多"这一规则在足够长的时间线上运行的必然结果。

### 分化过程

汗的血脉在立耳群体中逐代集中，立耳之间交配的后代进一步强化了相关性状。立耳与裂耳之间的差距逐渐从个体战斗力扩展为**体格、反应速度乃至体型上的可见差异**。立耳血脉的幼体出生时即可能呈现可辨识的特征——体型显著偏大、毛色异于常体、耳朵形态不同。

这一现象类似于犬类中牧羊犬或灵缇的品系分化——先天特征在出生时即可辨认。但穴兔的立耳后代仍须经历完整的淘汰流程，只是对他们而言淘汰几乎不构成悬念。

### 隐性种姓化

穴兔社会正从纯粹的实力淘汰制悄然滑向事实上的种姓制度，但穴兔自身对此缺乏认知。"任何个体都有机会凭实力攀升"的叙事仍在运转，而底层个体击败立耳后代的概率实际上已经趋近于零。

偶尔仍有不具备立耳血脉特征的底层个体凭借实力与运气打入顶层，这类个体在立耳群体中会格外引人注目。

### 失败的立耳后代

携带立耳血脉特征但在淘汰中表现平庸的个体将直接沦为底层储备粮，不享有任何特殊待遇。

### 父系积累

血脉分化完全沿父系路径积累——雄性一夫多妻制使得筛选压力集中于父系。母系仅为载体。汗的择偶不遵循任何优生原则，纯粹出于个体偏好。

### 以太的可能作用

立耳血脉的持续强化可能不仅是基因层面的筛选结果。高战斗力个体可能天然储存更高浓度的以太，导致肌腱密度、神经反应速度、骨骼强度等指标超出纯生物学可解释的范围。但穴兔自身对此毫无认知。

---

## 繁殖

- 汗为唯一享有多配偶特权的个体
- 其余个体无固定配偶关系，雌性在非妊娠期无法拒绝交配，整体呈无序混交状态
- **妊娠期保护机制**：妊娠中的雌性不会受到侵犯，且会获得食物供给。这并非社会性关怀，而是本能驱动的种群保护机制
- 母体照护持续约**一年半**，此后幼体进入淘汰流程
- 穴兔社会不存在家庭概念。亲代关系仅限于哺育期的供给关系，此后在社会层面与生理层面（气味识别）均不再维持
- 穴居环境对繁殖率存在天然上限，但相对其他知性物种仍属极高水平

## 生命周期

| 阶段 | 时间 | 说明 |
| ---- | ---- | ---- |
| 幼年期 | 0–6 个月 | 由母体照护，实际哺养延续至一年半 |
| 未成年期 | 6 个月–一岁半 | 以未成年杂鱼身份从事基础劳作，开始参与初级淘汰 |
| 活跃期 | 一岁半–十二岁 | 主要活动阶段，大部分个体在此期间死亡 |
| 自然寿命上限 | 约三十岁 | 仅在无外力致死的条件下（如被豢养个体）可达 |

这一快速的世代迭代意味着：工匠坑中的人类一代人可经历数任汗的更替；长寿的工匠头目比任何一只穴兔都存活更久，这是他们得以积累经验和影响力的根本原因。

东部被煌武驯化的族群中，个体普遍存活更长。更长的寿命带来更多的经验积累与更复杂的行为模式，与西部短寿、粗犷、以本能生存的族群形成了显著分化。

## 沟通

穴兔不具备文字系统，亦无系统性的知识传承机制。所有经验依靠个体在高强度淘汰中自行积累。

日常交流依赖三种渠道：

| 渠道 | 功能 |
| ---- | ---- |
| **动作** | 即时的行为指令与威胁信号 |
| **吼声** | 远距离警报与情绪表达 |
| **气味** | 个体识别、领域标记、等级辨认 |

其中气味可能承担着最核心的社会功能。立耳血脉分化后，不同等级之间的体味差异趋于明显。地道中残留的气味标记即为领域边界——无需专门巡逻，穴兔的体味本身即构成警报系统。

大部分穴兔没有名字。仅立耳和少数战功显著的裂耳会因战绩获得外号性质的称谓。绝大多数个体自始至终都是无名的。

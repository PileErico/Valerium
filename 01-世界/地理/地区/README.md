# 地区

各政区、领域与无政权区域的整体地理。详细方位导读见 [地理总览](../README.md)；下方大致由北至南排列。

- [诺德兰](诺德兰.md) — 北境，极境山脉以南的北方区域
- [格莱德尼亚](格莱德尼亚.md) — 大森林外圈的西北沿海
- [科鲁维亚](科鲁维亚.md) — 莱彻斯河南岸，自维里塔利亚分裂
- [科莱帕齐亚](科莱帕齐亚.md) — 莱彻斯河北岸，自维里塔利亚分裂
- [科克兰](科克兰.md) — 莱彻斯河北岸渡口，与厄里恩特隔河相望
- [厄里恩特](厄里恩特.md) — 德拉科尼斯以北的低海拔山地
- [德拉科尼斯](德拉科尼斯.md) — 帝国核心行省
- [阿戈隆德](阿戈隆德.md) — 德拉科尼斯以南的高原
- [雷加利亚](雷加利亚.md) — 帝国诸地中面积最大的富庶地区
- [奥达尼亚](奥达尼亚.md) — 拉斐涅平原所在
- [安塔利亚](安塔利亚.md) — 帝国西南半岛
- [阿奎莱亚](阿奎莱亚.md) — 帝国南部半岛
- [艾瑟利亚](艾瑟利亚.md) — 龙脊以东偏北的独立地区，非帝国开拓领
- [弗洛斯特](弗洛斯特.md) — 帝国南部开拓领
- [莫洛恩](莫洛恩.md) — 帝国中部开拓领
- [卡·文图斯](卡·文图斯.md) — 大陆中部北方
- [阿尔·姆哈](阿尔·姆哈.md) — 卡·文图斯以南的沙漠
- [煌武](煌武.md) — 大陆最东端
- [塔克兰群岛](塔克兰群岛.md) — 西南远海的群岛

# 煌武 · 地区

煌武全境划为九州，命名取「单字+州」的禹贡式质感，以煌武自身地理立名。完整的地理总述——大陆位置、地势、山脉、水系、奇观——见派系文档「煌武」。

## 九州

**泱水以北（边防低开发带）**

- 朔州 — 横贯北境、煌武最北；北方山脉加长垣，以驻军边防为主
- 垠州 — 西北边地；半干旱草原，向西北接卡·文图斯

**泱水以南（核心开发区）**

- 岚州 — 最西依天山；山前丘陵加东部沼泽，湿润多雾
- 嵚州 — 西南；天山南段与沧河源头，钧天九重圜丘所在
- 泱州 — 中央偏南；京畿，国都天枢所在
- 衍州 — 泱水诸汊汇集之地，永安渠治理后方宜居
- 浦州 — 东北；泱水下游入海口
- 陉州 — 两山夹一隘，南北咽喉
- 沧州 — 东南；沧河下游出海口，核心城沧阳（煌武人习惯称本地为「临沧」）

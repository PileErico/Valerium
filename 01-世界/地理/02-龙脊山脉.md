# 龙脊山脉

大陆西部最显著的地貌，南北走向，纵贯全境。帝国本土位于山脉以西，开拓领与其他势力位于以东。山脉上有若干可通行的隘口，是连通东西的关键交通节点。

## 走向与范围

北段与东西走向的**极境山脉**相接，极境山脉将诺德兰与赫里姆荒原分隔。

## 主要隘口

| 隘口 | 位置 | 连通 |
| ---- | ---- | ---- |
| **阿戈隆德隘口**【名称待定】 | 山脉南段，阿戈隆德控制 | 帝国本土 ↔ 弗洛斯特领（南部开拓领） |
| **德拉科尼斯北部隘口**【名称待定】 | 山脉中段 | 德拉科尼斯 ↔ 莫洛恩（中部开拓领） |

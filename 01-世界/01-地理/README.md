# 地理总览

大陆名为**瓦勒瑞姆（Valerium）**。世界观主要围绕大陆西部展开。

## 大陆格局

大陆从西到东分为三个主要板块：

- **西部**：龙脊山脉以西至海岸线。七冠诸国与帝国的活动范围几乎全部位于此处。
- **中部**：龙脊山脉以东。北方为卡·文图斯，南方为阿尔·姆哈，地形以草原、湿地、丘陵和沙漠为主。
- **东部**：煌武，大陆最东端的大国。

此外，帝国在龙脊山脉东侧建有两处**开拓领**，以及独立于帝国的**艾瑟利亚**也位于山脉东侧偏北。西南远海则有**塔克兰群岛**。

## 主要地貌

### 山脉

- **[龙脊山脉](02-龙脊山脉.md)**：大陆西部最显著的地貌，南北走向，纵贯全境。帝国本土位于山脉以西，开拓领与其他势力位于以东。山脉上有若干可通行的隘口，是连通东西的关键交通节点。
- **极境山脉**：东西走向，位于大陆西北部，与龙脊山脉北段相接。将诺德兰与赫里姆荒原分隔。
- **阿奎莱亚北部山地**：东西走向，将阿奎莱亚半岛与帝国腹地（雷加利亚、奥达尼亚）隔开。**灰雾隘口**是穿越此山地的主要通道，常年雾气弥漫。

### 隘口

| 隘口 | 位置 | 连通 |
| ---- | ---- | ---- |
| **灰雾隘口** | 阿奎莱亚北部山地 | 奥达尼亚 ↔ 阿奎莱亚 |
| **阿戈隆德隘口**【名称待定】 | 龙脊山脉南段，阿戈隆德控制 | 帝国本土 ↔ 弗洛斯特领（南部开拓领） |
| **德拉科尼斯北部隘口**【名称待定】 | 龙脊山脉中段 | 德拉科尼斯 ↔ 莫洛恩（中部开拓领） |

### 河流

- **莱彻斯河**：源自龙脊山脉，向西流淌。在厄里恩特附近出山，海拔骤降形成河谷平原（厄尔诺一带为重要粮仓）。主流横贯维里塔利亚中游，将其一分为二（南岸科鲁维亚、北岸科莱帕齐亚）。
- **雷加利亚河流**【名称待定】：雷加利亚境内的河流水系，为其西北部冲积平原提供灌溉。

### 林地

- **大森林**【名称待定】：大陆西部偏北的巨型林带，从莱彻斯河下游一直延伸至西部海岸。南北跨度极大，植被从北部针叶林渐变至南部橡木林。格莱德尼亚位于大森林外圈。

### 平原

- **拉斐涅平原**：雷加利亚以南、阿奎莱亚以北的开阔地带，奥达尼亚所在。是连接帝国腹地与南部半岛的必经之路。

### 极地

- **[赫里姆荒原（Hrímmark）](01-赫里姆荒原.md)**：极境山脉以北，寒冬永驻的冰原。

## 西部大陆 — 龙脊山脉以西

> 从北到南依次排列。

### [诺德兰（北境）](03-诺德兰-地理.md)

极境山脉以南的北方区域。"诺德兰"为书面称呼，"北境""北地"为口语。地广人稀，长年冰雪。

### [格莱德尼亚](04-格莱德尼亚-地理.md)

大森林外圈的西北沿海区域。南北有别。林中分布有多棵德鲁伊培育至巨大规模的圣橡树，各自对应一个部落的定居中心。

### [科鲁维亚](05-科鲁维亚-地理.md)

莱彻斯河南岸，从维里塔利亚分裂而来。

### [科莱帕齐亚](06-科莱帕齐亚-地理.md)

莱彻斯河北岸，从维里塔利亚分裂而来。

### [科克兰](07-科克兰-地理.md)

科莱帕齐亚东部，莱彻斯河北岸渡口，与厄里恩特隔河相望。

### [厄里恩特](08-厄里恩特-地理.md)

龙脊山脉以西、德拉科尼斯以北的低海拔山地。莱彻斯河在此出山。

### [德拉科尼斯](09-德拉科尼斯-地理.md)

龙脊山脉西侧、阿戈隆德高地以北及周围林地。兽龙类（drake）的主要栖息地与活动范围，因此得名。帝国核心行省。

### [阿戈隆德](10-阿戈隆德-地理.md)

德拉科尼斯以南的高原地区。控制着一处可穿越龙脊山脉的重要隘口，通往南部开拓领。

### [雷加利亚](11-雷加利亚-地理.md)

德拉科尼斯西南方、科鲁维亚以南。由冲积平原、森林与少量山地组成的辽阔富裕地区，面积在帝国诸地中最大。

### [奥达尼亚](12-奥达尼亚-地理.md)

拉斐涅平原上的地区，位于雷加利亚以南、阿奎莱亚以北。

### [安塔利亚](13-安塔利亚-地理.md)

帝国西南半岛。与雷加利亚西部相邻。

### [阿奎莱亚](14-阿奎莱亚-地理.md)

帝国南部半岛。与腹地之间隔着东西走向的山地，通过灰雾隘口连通。

## 龙脊山脉以东

### [艾瑟利亚](15-艾瑟利亚-地理.md)

龙脊山脉东侧偏北的独立地区，与卡·文图斯接壤。非帝国开拓领。

### [弗洛斯特领](16-弗洛斯特-地理.md)

帝国南部开拓领，位于龙脊山脉东侧南部。经阿戈隆德隘口（陆路）或阿奎莱亚海路（绕龙脊山脉南端）连通帝国本土。

### [莫洛恩](17-莫洛恩-地理.md)

帝国中部开拓领，位于龙脊山脉东侧中部。自德拉科尼斯北部穿越龙脊山脉。

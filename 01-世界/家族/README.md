# 家族

家族作为势力实体的条目——结构、生意、领地、政治位置与世代史。成员个人小传在角色文件夹中独立立档，此处只做家族层面的叙事。

## 目录

### 帝国四大家族【成员清点中】

- 欧恩斯坦家族 — 德拉科尼斯统治家族，龙裔血脉
- 奥利维亚家族 — 阿戈隆德，骑士与军官的摇篮，银鹫会中坚
- 梅特卢姆家族 — 学阀，学者与法师的摇篮，自称星民后裔
- 【待定】第四家待确认

### 帝国其他家族

- 维尔纳家族 — 厄里恩特渊源，公爵叛乱的策源
- 德拉布雷家族 — 维尔纳的下级家族，恩佐一支前来投奔
- 格兰蒂斯家族 — 科克兰武勋贵族，雇佣兵生意

### 南方（阿奎莱亚）

- 莱巴赫家族 — 联合会头号家族，萨莱诺港的主人
- 贝洛家族 — 卡利亚商业贵族，衰而复兴
- 科斯塔家族 — 费什海姆地下世界的经营者

### 北境

- 乌涅提斯氏族 — 奥列克的氏族，南迁蛮族谱系

# 命名规则

## 派系命名语系

| 语系 | 适用范围 | 说明 |
| ------ | ------ | ------ |
| 日耳曼语 | 德拉科尼斯及帝国核心 | 帝国发源地的语言传统 |
| 拉丁语 | 与德拉科尼斯同级的领地（日后降为公爵领） | 阿奎莱亚、维里塔利亚、雷加利亚等 |
| 凯尔特语/丹麦语 | 北地人 | 蛮族与北境 |
| 英语 | 阿戈隆德等被征服地 | |
| 阿拉伯语 | 阿尔·姆哈 | |

## 地名含义

七冠势力以拉丁语命名，-ia 为阴性后缀，表"某某之地/国度"。德拉科尼斯例外，使用拉丁语属格形式。

| 地名 | 英文 | 含义 |
| ------ | ------ | ------ |
| 德拉科尼斯 | Draconis | Drake=龙，拉丁语属格"龙的" |
| 雷加利亚 | Regalia | Regalis=王权。王权之地 |
| 维里塔利亚 | Veritalia | Veritas=真理。真理之地 |
| 阿奎莱亚 | Aquileia | 意大利现实地名，直接借用 |
| 艾瑟利亚 | Aetheria | Aether=以太。以太之地 |
| 格莱德尼亚 | Gladenia | Glade=林间空地。林间之地 |
| 奥达尼亚 | Ordania | Ordo=秩序。秩序之地 |
| 科鲁维亚 | Coruvia | 源自领主姓氏科鲁瓦（Coruva）+ -ia |
| 科莱帕齐亚 | Colepacia | 源自领主姓氏科莱帕兹（Colepaz）+ -ia |

# 第二次内战（血宴）

## 背景

皇帝（第一席）不甘被议会架空，渴望摆脱总帅[安德烈](../../角色/欧恩斯坦家族.md)的阴影，效仿先祖温德林建立绝对掌控的中央集权"新帝国"。

## 核心参与者

| 角色 | 身份 | 动机 |
| ------ | ------ | ------ |
| 皇帝（第一席） | 幕后主使 | 渴望中央集权，摆脱安德烈的阴影 |
| [艾德琳·欧恩斯坦](../../角色/欧恩斯坦家族.md)（第三席） | 总导演 | 看清皇帝野心，选择成为最锋利的刀，谋求新秩序核心地位 |
| [维克·欧恩斯坦](../../角色/欧恩斯坦家族.md)（第五席） | 秘密合伙人 | 乐见安德烈陨落，换取皇室对魔导装备项目的全力支持 |
| [安德烈·欧恩斯坦](../../角色/欧恩斯坦家族.md)（第二席） | 目标 | 早已洞悉一切，选择赴死 |
| [泽娜·欧恩斯坦](../../角色/欧恩斯坦家族.md)（第六席） | 被利用的棋子 | 安德烈亲手制造的生物兵器，被艾德琳用作真正的胜负手 |

## 阴谋三层嵌套

### 表层——阳谋："叛乱"

艾德琳利用外交官身份，秘密联络帝国境内所有对中央集权不满的地方领主和宫廷贵族，许诺一个"宏大"的叛乱计划：

- **外部**：各地领主同时起兵，分散帝国兵力
- **内部**：由宫廷贵族作为内应，在安德烈离京后发动政变
- **最终目标**：内外夹击围歼安德烈，建立权力下放的"新议会"

**真实目的**：动员那些手握兵权但生性多疑的地方领主，让他们成为未来内战的主力炮灰。

### 核心——阴谋："刺杀"

艾德琳向帝都内一小部分无兵权但极度渴望权力的"内应"（宫廷贵族、商人、无地领主）透露"真正计划"：

- 揭示泽娜这张"生物兵器"底牌
- 煽动他们："我们不必等待漫长的战争，只要在宴会上一举杀死安德烈，就能抢占头功，挟持皇帝，一步登天！"

**真实目的**：利用这群人的贪婪和天真，让他们成为引爆事件的"消耗品"。与安德烈相比，这些人只不过是用来送死和拖延时间的，真正的胜负手是泽娜。

### 最终——战略："大清洗"

皇帝和艾德琳真正的目标，一石三鸟：

1. **借刀杀人**：借叛军之手，名正言顺地除掉安德烈
2. **借安德烈之手清除异己**：利用安德烈的强大反过来清除所有激进派和投机派贵族，完成对宫廷的血腥清洗
3. **树立权威**：皇帝以"平定叛乱、为总帅复仇"的救世主姿态登场，为后续集权铺路

更深一层：安德烈杀死这群人和摇摆派之后，整个宫廷暂时瘫痪陷入空虚。艾德琳的人名正言顺接手他们的遗产，而外部叛逆领主失去内应，将逐渐沦为被动。

## 血宴之夜

> 详细场景描写见 [血宴之夜（短篇）](../../故事/短篇与片段/帝国03-血宴之夜.md)

1. **激进派暴起**——被艾德琳煽动的宫廷贵族率先攻击安德烈。安德烈徒手将其全部屠灭
2. **投机派转向**——见安德烈不可敌，投机派转而攻击泽娜以表明立场。泽娜龙人化，同样将其屠灭
3. **毒与对峙**——艾德琳在酒中下明毒为诱饵，真正的毒藏在菜肴中。安德烈识破酒毒却未防备菜肴。艾德琳与安德烈进行最后的理念对峙
4. **军神落幕**——毒素发作后安德烈与泽娜死斗。安德烈放弃反杀，临终抚摸泽娜头顶后靠在她肩上死去

## 安德烈的选择

安德烈早已洞悉一切但不阻止：

- 战友和亲人百年前全部死去，独自守护帝国两百余年已疲惫
- 相信年轻一代必须脱离庇护才能成长
- 渴望一场盛大的落幕——不是默默老死，而是以总帅身份、在帝国心脏、以战斗姿态结束
- 他甘赴鸿门宴，是出于对自己人生的疲惫、对帝国未来的最后考验、以及对一场荣耀战死的渴望

## 战后：掌控叙事

- **封锁消息**：皇帝将事件定性为"少数野心家发动的叛乱，总帅不幸遇难，皇室平定叛乱"
- **召回铎利克**：以"为恩师复仇"的大义名分，邀其出任新总帅，领军清剿响应叛乱的外部同党（即被艾德琳卖掉的地方领主）
- **洛德被定罪**：被定义为"叛党核心成员之一"，与奥列克一同列为帝国首要铲除目标
- **[北境](../文明/帝国/北境.md)借机独立**：奥列克趁帝国内乱推动北境脱离
- **泽娜出走**：离开德拉科尼斯开始流浪

皇帝以"为安德烈复仇、肃清帝国"为旗号发起战争，真正目的是清除所有反对集权的地方势力。被蒙在鼓里的洛德与铎利克，为给老师复仇不得不与打着同样旗号的帝国大军兵戎相见——这就是第二次内战最大的悲剧。

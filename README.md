# 命运长歌 Valerium

一个原创奇幻世界观企划。

旨在创造一个尽可能兼顾真实感且带有奇幻要素的世界观，涵盖从蛮荒直到中世纪末期的战争与信仰变革。通过描绘派系间的对抗以及个人的冒险，分别从宏观和微观角度展现这个世界的一系列变迁。

这是一个涵盖了剑与魔法、唯物主义魔法、唯心主义奇迹的世界。

尽管这个题材很难回避同质化的问题，但建议不要带着刻板印象去看待这里的大部分东西。你**不会**在这里找到千娇百媚的精灵，郊游嬉戏的冒险者团体，宛如儿戏的军事对抗，以及总是被当成反派角色塑造的教廷。

如果没有异议，那就继续往下看吧。因为设定比较繁杂，推荐在目录中挑选自己**感兴趣**的内容阅读，无需考虑先后顺序。

# 赏金猎人

## 起源

帝国建立初期，为逐步推行统一律法、加强对地方管制，建立了**巡回法庭**——由教士、学士和宫廷骑士组成的流动仲裁团体。所有重大犯罪的嫌疑人都需由巡回法庭审判，地方贵族只能羁押候审，不可擅自定罪。

但巡回法庭效率较低，大部分地区会产生无人过问的空窗期。为减少长期关押的额外开支，地方贵族通常允许嫌疑人保释。为避免保释人在候审期间逃亡，独属于赏金猎人的**定向委托**由此诞生。

## 两类委托

- **定向委托**：监管并确保保释嫌疑人出席庭审，一旦嫌疑人逃离则展开追捕。报酬为保释金两成。
- **自由委托**：以公开悬赏方式进行，委托人提出需求并确保能支付相当的报酬。秉承先到先得原则，任何人可尝试接取，但只有先一步证明完成委托的猎人才获得奖励。

## 鬣狗

以猎人为目标的猎人，抢夺他人成果。另见[鬣狗人种族](../01-世界基础/03-种族总览.md)。

## 分级（非官方，同行口碑）【待重做】

赏金猎人没有规范化的管理和登记，信用和能力只能通过同行口耳相传来印证。以下为当前的非正式分级：

| 等级 | 说明 |
| ------ | ------ |
| Newbie（菜鸟） | 入行新人，死亡率极高 |
| Veteran（合格） | 入行一年仍活跃在一线，算是摆脱了菜鸟范畴 |
| Expert（专家） | 完成极难委托、展现他人所不能之事。到此通常会被贵族或商人招揽，不再全职 |
| Elite（精英） | 行业顶端，几乎无缺陷，每方面都做到最好，要价极贵。连王公贵族也会指名委托 |

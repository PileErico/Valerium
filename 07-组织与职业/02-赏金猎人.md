# 赏金猎人

## 起源

帝国巡回法庭效率低产生空窗期，保释制度催生定向委托。

## 两类委托

- **定向委托**：监管保释嫌疑人，确保出席庭审。报酬为保释金两成。
- **自由委托**：公开悬赏，先到先得。

## 鬣狗

以猎人为目标的猎人，抢夺他人成果。

## 分级（非官方，同行口碑）

| 等级 | 说明 |
|------|------|
| Newbie（菜鸟） | 入行新人 |
| Veteran（合格） | 入行一年仍活跃 |
| Expert（专家） | 完成极难委托，通常被招揽离开一线 |
| Elite（精英） | 行业顶端，几乎无缺陷，要价极贵 |

## 无管理体制

完全自由不受管制，死亡率极高，信用能力全靠口碑。
